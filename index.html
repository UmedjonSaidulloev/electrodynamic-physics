Қонуни бақои заряди электрикӣ

Назарияи асосӣ

Қонуни бақои заряди электрикӣ изҳор медорад, ки заряди умумии як система дар ҳама гуна раванди физикӣ тағйир намеёбад. Зарядҳо наметавонанд худ аз худ ба вуҷуд оянд ё нобуд шаванд.

Моҳиятҳои асосӣ:

  • Зарядҳо метавонанд байни ҷисмҳо тақсим шаванд, аммо суммаи умумии онҳо доимист
  • Қонун дар муҳити баста (бе таъсири беруна) пурра иҷро мешавад
  • Дар тамоми физика — аз механика то физикаи квантӣ — татбиқ мегардад

Формулаи асосӣ

Σq = const

Яъне, суммаи умумии ҳамаи зарядҳо дар система тағйирнопазир мемонад.

Шарҳ:

Қонуни бақои заряд асоси назариявии фаҳмиши бисёре аз равандҳои электромагнитӣ мебошад. Он ба системаҳое, ки дар он табодули заряд сурат мегирад, ба монанди реаксияҳои химиявӣ, таъсири майдонҳо ва плазмаҳо низ татбиқ мешавад.

Таҷрибаи назариявӣ

Таъҷриба: Агар ду ҷисм бо зарядҳои +5μC ва -2μC бо ҳам тамос гиранд, пас аз тақсимшавии зарядҳо, суммаи умумии заряд бояд баробар бо 3μC боқӣ монад.

Мисол: Агар пас аз тамос зарядҳо ба 1μC ва 2μC тақсим шаванд:

Σq = 1μC + 2μC = 3μC → қонун иҷро мешавад

Масъалаҳои намунавӣ

Масъала 1 (Содда): Ду зарра бо зарядҳои +3μC ва -3μC. Оё қонуни бақои заряд иҷро мешавад?

Ҳал:

Σq = 3μC + (-3μC) = 0 → Қонун иҷро мешавад

Масъала 2 (Миёна): Ҷисме бо заряди +4μC ба ду ҷисми дигар тақсим шуд. Яке аз онҳо 1.5μC аст. Заряди дуюмро ёбед.

Ҳал:

q₂ = 4μC - 1.5μC = 2.5μC

Σq = 1.5μC + 2.5μC = 4μC → Қонун иҷро мешавад

Масъала 3 (Мураккаб): Се зарра бо зарядҳои 6μC, -3μC ва xμC мавҷуданд. Баъд аз тағйирот онҳо ба 2μC, -4μC ва yμC мубаддал мешаванд. Арзиши x ва y-ро ёбед.

Ҳал:

Пеш: Σq₁ = 6 + (-3) + x = 3 + x

Баъд: Σq₂ = 2 + (-4) + y = -2 + y

Баробар мекунем: 3 + x = -2 + y → y = x + 5μC

Масалан: x = 1μC → y = 6μC

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди +6μC ба ду зарраи дигар тақсим мешавад. Агар яке 2.3μC бошад, дуввумро ёбед ва санҷед, ки қонун иҷро мешавад.

2. Се зарра бо зарядҳои +2μC, -5μC ва xμC мавҷуданд. Агар пас аз як таъсир онҳо ба +1μC, -4μC ва yμC мубаддал шаванд, арзишҳои x ва y-ро ёбед.

3. Ду ҷисм бо зарядҳои -3μC ва -2μC бо ҳамдигар тамос мегиранд ва зарядҳо баробар тақсим мешаванд. Заряди ҳар як пас аз тамосро ёбед.

4. Ҷисме бо заряди 10μC ба се қисм тақсим мешавад. Агар дутои онҳо 3μC ва 4.5μC бошанд, севумро ҳисоб кунед ва бо Σq = const санҷед.

5. Дар система зарядҳо: +4μC, xμC ва -3μC мавҷуданд. Агар зарядҳои нав пас аз таъсир чунин бошанд: +1μC, yμC ва -2μC, пас арзишҳои x ва y-ро ёбед.

Қонуни Кулон

Назарияи асосӣ

Қонуни Кулон мегӯяд, ки байни ду зарраи барқнок қуввае пайдо мешавад, ки ба ҳосили зарядҳои онҳо мутаносиб ва ба квадрати масофа баръакс мутаносиб аст.

Қиматҳои доимӣ:

  • k = 9×10⁹ N·m²/C² — доимии Кулон
  • ε₀ = 8.85×10⁻¹² C²/N·m² — доимии электрикӣ дар вакуум

Формулаи асосӣ

F = k·(q₁·q₂)/r²

F — қувваи электростатикӣ; q₁ ва q₂ — зарядҳо; r — фосилаи байни онҳо

Шарҳ:

Қонуни Кулон танҳо барои зарраҳои нуктавӣ (ё зарраҳое, ки масофа байни онҳо аз андозаи худи онҳо хеле калон аст) истифода мешавад.

Ин қонун асоси электростатика мебошад ва муайян мекунад, ки:

  • Агар зарядҳо якхела бошанд — онҳо якдигарро тела медиҳанд
  • Агар зарядҳо муқобил бошанд — онҳо якдигарро мекашанд

Дар аниматсия дида мешавад, ки қувваи байни ду зарра ҳамеша ба қонуни Кулон итоат мекунад, новобаста аз мусбат ё манфӣ будани онҳо. Қувваҳо баробар ва ба самтҳои муқобил мебошанд:

|F₁| = |F₂| = k · |q₁ × q₂| / r²

Таҷрибаи назариявӣ

Таъҷриба: Ду зарра бо зарядҳои +1μC ва -1μC дар фосилаи 10 см ҷойгиранд.

Ҳисоб: F = (9×10⁹)·(1×10⁻⁶)(1×10⁻⁶)/(0.1)² = 0.9 N

Ин қувваи кашиши электростатикӣ бо таҷриба (масалан бо электрометр) чен карда мешавад.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Ду зарра бо зарядҳои +2μC ва +2μC дар фосилаи 30 см ҷойгиранд. Қувваро ёбед.

Ҳал:

F = (9×10⁹)·(2×10⁻⁶)(2×10⁻⁶)/(0.3)² = 0.4 N

Масъала 2 (Миёна): Ду зарраи бо зарядҳои 5μC ва 5μC дар фосилаи 20 см ҷойгиранд. Қувваро ёбед.

Ҳал:

F = (9×10⁹)·(5×10⁻⁶)²/(0.2)² = 5.625 N

Масъала 3 (Мураккаб): Се зарра бо зарядҳои 2μC, -3μC ва 1μC дар як хат ҷойгиранд. Фосилаи аввал ва дуюм — 10 см, дуюм ва сеюм — 20 см. Қувваи умумӣ ба зарраи дуюмро ёбед.

Ҳал:

F₁₂ = (9×10⁹)·(2×10⁻⁶)(3×10⁻⁶)/(0.1)² = 5.4 N (кашиш)

F₃₂ = (9×10⁹)·(1×10⁻⁶)(3×10⁻⁶)/(0.2)² = 0.675 N (кашиш)

F = 5.4 N + 0.675 N = 6.075 N

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду зарра бо зарядҳои +4μC ва -4μC дар фосилаи 15 см ҷойгиранд. Қувваи байни онҳоро бо ёрии қонуни Кулон ҳисоб кунед.

2. Зарядҳои +2μC ва +6μC дар масофаи 25 см ҷойгиранд. Қувваи байни онҳоро ёбед ва самти таъсири он ба ҳар як зарраро муайян кунед.

3. Дар вакуум ду зарра бо зарядҳои q₁ = -3μC ва q₂ = -5μC ҷойгиранд. Масофа байни онҳо 40 см аст. Қувваи байни онҳоро ҳисоб кунед ва навъи таъсирро шарҳ диҳед.

4. Заряди 8μC ба ду зарраи дигар бо зарядҳои 3μC ва 5μC баробар масофа дорад. Фарзи фосила 10 см бошад. Ҳисоб кунед, ки ҳар як зарра бо чӣ қувва таъсир мекунад.

5. Се зарра дар як хат ҷойгиранд: +2μC, -1μC ва +3μC. Масофа байни ҳар як ҷуфт 20 см аст. Қувваи умумӣ ба зарраи миёна (–1μC)-ро бо усули векторӣ ҳисоб кунед.

Майдони электростатикӣ ва шадидияти он

Назарияи асосӣ

Майдони электрикӣ фазое мебошад, ки дар он ҳар як заряд таъсири электрикӣ ҳис мекунад. Он аз тарафи зарядҳои электрикӣ ба вуҷуд меояд ва дар ҳар як нуқтаи фазо дорои қувва ва самт аст.

Хусусиятҳои майдони электрикӣ:

  • Дар атрофи ҳар як заряд майдон вуҷуд дорад
  • Майдони электрикӣ векторӣ мебошад (самт ва арзиш дорад)
  • Барои муайян кардани он, зарядҳои санҷишӣ истифода мешаванд

Формулаҳои асосӣ

E = F / q

Шадидияти майдон баробари таносуби қувва ба заряд мебошад.

E = k · (q / r²)

Шадидияти майдон аз тарафи як заряд дар масофаи r

Шарҳ:

Майдони электрикӣ мисли қувваи гравитатсионӣ амал мекунад, вале байни зарядҳои электрикӣ. Самти он вобаста ба навъи заряд муайян мешавад:

  • Агар заряд мусбат бошад — хатҳои майдон аз он берун мешаванд
  • Агар заряд манфӣ бошад — хатҳои майдон ба он ворид мешаванд

Дар аниматсия дида мешавад, ки майдон боиси таъсир ба дигар зарядҳо мешавад ва ҳамеша дар нуқта дорои самт ва арзиши мушаххас аст.

Таҷрибаи назариявӣ

Таъҷриба: Ду заряд — яке мусбат, дигаре манфӣ. Агар дар байни онҳо заряди хурд гузошта шавад, самти ҳаракати он самти майдонро нишон медиҳад.

Ҳисоб: Агар q = 2μC ва F = 0.02 N →

E = F / q = 0.02 / (2×10⁻⁶) = 10,000 N/C

Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряди 1μC дар майдон қувваи 0.003 N эҳсос мекунад. Арзиши майдонро ёбед.

Ҳал:

E = F / q = 0.003 / (1×10⁻⁶) = 3,000 N/C

Масъала 2 (Миёна): Заряди 2μC дар фосилаи 30 см. Чӣ қадар майдон тавлид мекунад?

Ҳал:

E = k·q / r² = (9×10⁹)·(2×10⁻⁶) / (0.3)² = 2×10⁵ N/C

Масъала 3 (Мураккаб): Ду заряд 3μC ва -3μC дар фосилаи 40 см. Дар нуқтаи миёна шадидияти майдон чӣ қадар мешавад?

Ҳал:

Фосила то ҳар заряд r = 0.2 м. Самтҳо якхела, қиматҳо баробар:

E = 2 · k · (3×10⁻⁶) / (0.2)² = 2 · 9×10⁹ · 3×10⁻⁶ / 0.04 = 1.35×10⁶ N/C

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди 2μC дар майдони электрикӣ бо шадидияти 4000 N/C қарор дорад. Қувваи таъсиркардаро ба ин заряд ҳисоб кунед.

2. Майдони электрикӣ аз тарафи заряди нуқтавии +5μC дар фосилаи 20 см таъсир мекунад. Арзиши шадидияти майдонро ҳисоб кунед.

3. Дар нуқтае, ки заряд q = -3μC қарор дорад, шадидияти майдон E = 1500 N/C аст. Қувваи барқӣ ба ин зарядро бо самташ ҳисоб кунед.

4. Заряди 6μC дар фосилаи 10 см майдони 5×10⁴ N/C тавлид мекунад. Қимати доимии Кулонро истифода бурда, шадидиятро ҳисоб кунед ва бо формулаи умумӣ тафтиш кунед.

5. Ду заряд бо +2μC ва -2μC дар фосилаи 30 см ҷойгиранд. Дар нуқтаи миёна байни онҳо чӣ қадар шадидияти майдон ба амал меояд ва самти он чӣ гуна аст?

Принсипи суперпозитсияи майдонҳои электрикӣ

Назарияи асосӣ

Принсипи суперпозитсия мегӯяд, ки агар дар як нуқтаи фазо якчанд заряд таъсир расонанд, пас майдони умумӣ баробари ҷамъбасти вектории ҳамаи майдонҳои алоҳидаи зарядҳо мебошад.

Хусусиятҳои асосӣ:

  • Ҳар як заряд дар нуқта майдони худро эҷод мекунад
  • Майдони умумӣ = суммаи вектории ҳамаи майдонҳои алоҳида
  • Қувва дар нуқта = қувваҳои алоҳида ҷамъ мешаванд

Формулаҳои асосӣ

Etotal = E₁ + E₂ + E₃ + ... + En

Агар майдонҳо ба як самт бошанд — ҷамъ мешаванд, агар муқобил — тарҳ мешаванд

Шарҳ:

Ин принсип асоси ҳисобкунии майдонҳои электрикӣ дар системаҳои серзарядро ташкил медиҳад. Азбаски майдон вектор аст, дар ҳолатҳои фазоӣ бояд усулҳои векториро истифода бурд: (тангент, синус, косинус, координатаҳо).

Таҷрибаи назариявӣ

Таъҷриба: Ду заряди мусбат дар ду нуқтаи гуногун гузошта шаванд. Бо ҷисми озмоишӣ (заряди хурд) ҳаракат ва самти майдони умумиро мушоҳида мекунем.

Ҳисоб: Агар E₁ = 1000 N/C ва E₂ = 1500 N/C, ва онҳо дар як самт бошанд:

Etotal = E₁ + E₂ = 2500 N/C

Масъалаҳои намунавӣ

Масъала 1 (Содда): Ду майдон бо арзишҳои 500 N/C ва 300 N/C дар як самт таъсир мерасонанд. Майдони умумиро ёбед.

Ҳал:

Etotal = 500 + 300 = 800 N/C

Масъала 2 (Миёна): Ду майдон бо арзишҳои 600 N/C ва 400 N/C дар самтҳои муқобил таъсир мерасонанд. Майдони умумиро ёбед.

Ҳал:

Etotal = 600 - 400 = 200 N/C (ба самти майдони калонтар)

Масъала 3 (Мураккаб): Ду заряд E₁ = 1200 N/C ва E₂ = 1600 N/C дар кунҷи 90° таъсир мекунанд. Майдони умумиро бо роҳи векторӣ ҳисоб кунед.

Ҳал:

Азбаски кунҷ 90° аст, ҷамъбаст бо теоремаи Пифагор:

Etotal = √(E₁² + E₂²) = √(1200² + 1600²) = √(1440000 + 2560000) = √4000000 = 2000 N/C

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду майдон бо арзишҳои 1200 N/C ва 800 N/C ба як самт таъсир мерасонанд. Майдони умумиро ҳисоб кунед.

2. Агар E₁ = 2000 N/C ва E₂ = 1500 N/C бошанд ва самтҳояшон муқобил бошад, майдони умумиро бо самташ муайян кунед.

3. Ду майдон бо E₁ = 1000 N/C ва E₂ = 1000 N/C дар кунҷи 60° таъсир мекунанд. Майдони умумиро бо усули векторӣ ҳисоб кунед.

4. Се майдон бо арзишҳои E₁ = 300 N/C, E₂ = 400 N/C ва E₃ = 500 N/C дар як самт таъсир мерасонанд. Шадидияти майдони умумиро ёбед.

5. Ду заряд бо q₁ = +2μC ва q₂ = -2μC дар фосилаи 20 см ҷойгиранд. Дар нуқтаи миёна байни онҳо шадидияти умумии майдонро ҳисоб кунед ва самти он чист?

Теоремаи Гаусс дар вакуумӣ

Назарияи асосӣ

Теоремаи Гаусс муносибати байни ҷараёни майдони электрикӣ ва заряди дохилӣ дар сатҳи пӯшидашударо тавсиф мекунад. Он яке аз теоремаҳои асосии электростатика мебошад.

Моҳият:

  • Ҷараёни электрикӣ аз рӯи сатҳи пӯшида мутаносиб ба заряди дохилӣ аст
  • Ба муҳити холӣ (вакуум) татбиқ мешавад
  • Ба ҳисобкунии майдонҳои симметрӣ (сфера, цилиндр, сатҳи ҳамвор) кӯмак мекунад

Формулаи асосӣ

∮E·dS = q / ε₀

Ҷараёни майдон (флюкс) баробари заряди дохилӣ тақсим бар доимии электрикӣ мебошад.

E = q / (4·π·ε₀·r²)

Барои майдони электрикӣ дар сатҳи сферӣ

Шарҳ:

Теоремаи Гаусс асоси назариявии ҳисобкунии майдонҳои электрикӣ дар ҳолати симметрии геометрӣ мебошад. Он ба шаклҳои оддии геометрӣ, ба монанди сфераҳо ва цилиндрҳо хуб татбиқ мешавад ва ҳисобкуниро содда мекунад.

Таҷрибаи назариявӣ

Таъҷриба: Ба тӯби заряднок варақи металлӣ наздик карда мешавад. Ҳангоми ченкунии майдон бо электрометр, тағйирёбии флюкс нисбат ба тағйири заряди дохилӣ мушоҳида мешавад. Тағйироти заряд, бо тағйироти флюкс, мутаносиб аст.

Мисол: Агар q = 1μC ва ε₀ = 8.85×10⁻¹², пас:

Флюкс = q / ε₀ = (1×10⁻⁶) / (8.85×10⁻¹²) ≈ 1.13×10⁵ N·m²/C

Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряди q = 2μC дар дохили сатҳи пӯшида ҷойгир аст. Флюкси электрикӣ аз рӯи ин сатҳро ёбед.

Ҳал:

Φ = q / ε₀ = (2×10⁻⁶) / (8.85×10⁻¹²) ≈ 2.26×10⁵ N·m²/C

Масъала 2 (Миёна): Майдони электрикӣ дар сатҳи сферии радиуси 10 см баробари 1.8×10⁵ N/C аст. Заряди дохилиро ёбед.

Ҳал:

Пеш аз ҳама, флюкс = E·S = E·4πr²

Φ = 1.8×10⁵ · 4π·(0.1)² ≈ 22620 N·m²/C

Пас: q = Φ·ε₀ = 22620 · 8.85×10⁻¹² ≈ 2×10⁻⁷ C

Масъала 3 (Мураккаб): Се зарра бо заряди 1μC, -2μC ва 0.5μC дар дохили сатҳи пӯшида ҷойгиранд. Флюкси умумиро ҳисоб кунед.

Ҳал:

Σq = 1 + (-2) + 0.5 = -0.5 μC = -0.5×10⁻⁶ C

Φ = q / ε₀ = (-0.5×10⁻⁶) / (8.85×10⁻¹²) ≈ -5.65×10⁴ N·m²/C

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди 3μC дар дохили сатҳи пӯшида ҷойгир аст. Флюкси электрикӣ аз рӯи ин сатҳро ҳисоб кунед (ε₀ = 8.85×10⁻¹² C²/N·m²).

2. Дар дохили сатҳи сферӣ, майдони электрикӣ E = 2×10⁴ N/C ва радиуси сатҳ 0.2 м мебошад. Флюкс ва заряди дохилиро ҳисоб кунед.

3. Дар як сатҳи пӯшида, флюкси электрикӣ Φ = 1.13×10⁵ N·m²/C чен карда шуд. Заряди дохилиро ёбед.

4. Майдони электрикӣ дар сатҳи силиндрӣ бо радиуси 0.1 м ва баландии 0.5 м баробар ба 5000 N/C аст. Флюксро ёбед, агар майдон ба таври яксон ба рӯйи паҳлӯ таъсир кунад.

5. Се зарра бо зарядҳои +2μC, -3μC ва +1μC дар дохили сатҳи пӯшида ҷойгиранд. Флюкси умумии майдони электрикӣ аз рӯи ин сатҳро ҳисоб кунед.

Татбиқи теоремаи Гаусс барои симметрияҳои гуногун

Назарияи асосӣ

Теоремаи Гаусс метавонад барои ҳисоб кардани майдони электрикӣ дар ҳолатҳое истифода шавад, ки симметрияҳои геометрӣ (сферӣ, силиндрӣ ё ҳамвор) мавҷуданд.

Навъҳои симметрияҳо:

  • Симметрияи сферӣ: майдон аз марказ берун меравад, E = q / (4πε₀r²)
  • Симметрияи силиндрӣ: майдон аз меҳвари силиндр радиалӣ берун меравад
  • Симметрияи ҳамвор (плоский): майдон дар ҳар ду самт якхела паҳн мешавад

Формулаҳои умумӣ

∮E·dS = q / ε₀

Барои ҳар як навъи симметрия, сатҳи Гауссро мувофиқи шакл интихоб мекунем.

Шарҳ:

Теоремаи Гаусс дар ҳолатҳое, ки майдони электрикӣ дар ҳама нуқтаҳои сатҳ якхела аст ва самти он ба сатҳ перпендикуляр аст, хеле самаранок аст. Ин боиси соддагардонии ҳисобҳо мегардад.

Таҷрибаи назариявӣ

Мисол: Агар заряди q дар маркази тӯб бошад, пас майдон дар сатҳи сферӣ якхела тақсим мешавад. Мо метавонем ба воситаи электрометр дар нуқтаҳои гуногун қувваро чен кунем ва тасдиқ кунем, ки майдон танҳо аз масофа вобаста аст.

Формула: E = q / (4·π·ε₀·r²)

Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряди нуқтавӣ q = 1μC дар маркази сфера ҷойгир аст. Радиуси сатҳи сферӣ 20 см. Майдони электрикӣ дар сатҳро ёбед.

Ҳал:

E = q / (4πε₀r²) = (1×10⁻⁶)/(4π·8.85×10⁻¹²·(0.2)²) ≈ 2.25×10⁵ N/C

Масъала 2 (Миёна): Заряди λ = 3×10⁻⁶ C/m дар як ноқили бениҳоят дароз мавҷуд аст. Майдони электрикӣ дар фосилаи 0.1 м аз меҳварро ёбед.

Ҳал:

Симметрияи силиндрӣ:

E = λ / (2πε₀r) = (3×10⁻⁶)/(2π·8.85×10⁻¹²·0.1) ≈ 5.4×10⁵ N/C

Масъала 3 (Мураккаб): Ду сатҳи параллел бо зичии сатҳии заряд σ = 5×10⁻⁶ C/m² зарядноканд. Майдони байни онҳо чӣ қадар хоҳад буд?

Ҳал:

Ҳар як сатҳ майдони E = σ / (2ε₀) тавлид мекунад. Дар байни онҳо майдонҳо ҷамъ мешаванд:

Etotal = σ / ε₀ = (5×10⁻⁶)/(8.85×10⁻¹²) ≈ 5.65×10⁵ N/C

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди нуқтавии +5μC дар маркази сатҳи сферӣ бо радиуси 0.3 м ҷойгир аст. Флюкси майдонро ҳисоб кунед ва шарҳ диҳед, ки оё радиус ба натиҷа таъсир мерасонад.

2. Майдони электрикӣ дар берун аз силиндри дароз, ки заряд дар дарунаш ҷойгир аст, баробар ба 4000 N/C мебошад. Агар радиуси силиндр 0.1 м ва баландӣ 1 м бошад, флюкс ва заряди дохилиро ҳисоб кунед.

3. Дар майдони электрикӣ ба шакли ҳамвор бо E = 2.5×10⁴ N/C ду лавҳаи калон бо зарядҳои муқобил ҷойгиранд. Заряди рӯизаминиро барои ҳар як лавҳа ҳисоб кунед.

4. Дар дохили сферои заряди мутамарказ флюкси умумӣ Φ = 3.39×10⁵ N·m²/C чен карда шуд. Заряди дохилиро бо истифода аз ε₀ = 8.85×10⁻¹² C²/N·m² ҳисоб кунед.

5. Барои силиндри симметрӣ бо радиуси 0.05 м ва баландии 0.5 м, флюкси майдон баробар ба 1×10⁴ N·m²/C аст. Заряди дохилии онро ҳисоб кунед ва нишон диҳед, ки қонун иҷро мешавад.

Сиркулятсияи вектори шадидияти майдон

Назарияи асосӣ

Сиркулятсияи вектори майдони электрикӣ ҷамъбасти скалярии зарби вектори майдон ба унсури масир (dl) дар як контури пӯшида мебошад. Дар электростатика, сиркулятсияи майдони электрикӣ ҳамеша 0 аст.

Моҳият:

  • Дар майдони электростатикии холис, сиркулятсия ҳамеша сифр аст
  • Сиркулятсия = кори майдон дар масири пӯшида
  • Ин қонуни муҳими майдонҳои потенсиалӣ мебошад

Формулаи асосӣ

∮𝐄·d𝐥 = 0

Сиркулятсияи майдони электрикӣ дар ҳолати электростатика = 0

A = -∫𝐄·d𝐥

Кор дар байни ду нуқта ҳамчун интеграли вектории шадидият

Шарҳ:

Расм татбиқи теоремаи Гаусс-ро дар муҳити симметрии силиндрӣ нишон медиҳад. Майдони электрикӣ \(\vec{E}\) ба сатҳи паҳлуии силиндр перпендикуляр буда, дарозии он \(\Delta l\) ва радиусаш \(R\) мебошад. Тибқи теоремаи Гаусс:

\[\oint_S \vec{E} \cdot d\vec{S} = \frac{q_{\text{in}}}{\varepsilon_0}\]

Ин ифода имкон медиҳад, ки шиддати майдони электрикӣ тавассути заряди дохили силиндр ҳисоб карда шавад.

Таҷрибаи назариявӣ

Таъҷриба: Як зарядро дар майдон ба масири пӯшида (масалан, давра) ҳаракат медиҳем. Бо асбоби сенсорӣ чен мекунем, ки кори умумӣ 0 мешавад. Ин ба сифр будани сиркулятсия мувофиқат мекунад.

Ҳисоб: Агар заряд q = 1μC бошад ва масир пӯшида, пас A = 0, яъне ∮𝐄·d𝐥 = 0

Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряде дар як майдони электрикӣ дар масири пӯшида ҳаракат мекунад. Кор = 0. Сиркулятсия чанд аст?

Ҳал:

∮𝐄·d𝐥 = A/q = 0/1μC = 0

Сиркулятсия = 0

Масъала 2 (Миёна): Заряди 2μC аз нуқтаи A ба нуқтаи B ҳаракат мекунад. Масири ҳаракат – рост, E = 1000 N/C, масофа = 0.05 м. Кор ва сиркулятсияро ёбед.

Ҳал:

Кор: A = q·E·d = 2×10⁻⁶·1000·0.05 = 0.0001 J

Сиркулятсия (танҳо дар контури пӯшида = 0)

Масъала 3 (Мураккаб): Барои заряди 1μC дар як майдони консервативӣ масири A → B → C → A ҳисоб кунед. Дар ҳар як масир E ва d маълум аст. Дар охир сиркулятсияро муайян кунед.

Ҳал:

Чун масир пӯшида аст (A → B → C → A), сиркулятсия ҳамеша:

∮𝐄·d𝐥 = 0

Новобаста аз арзишҳои миёнаи E ва d

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар як майдони электростатикӣ, дар масири пӯшида, шадидияти майдон E = 200 N/C мебошад. Сиркулятсияи майдон дар ин масирро ҳисоб кунед.

2. Вектори шадидияти майдон \(\vec{E}\) ба масири гардишӣ перпендикуляр аст. Далел оред, ки сиркулятсия баробар ба сифр аст ва чаро.

3. Дар минтақае шадидияти майдон E тағйирёбанда аст. Агар дар як контур 4 нуқта бо арзишҳои E = {100, 150, 130, 120} N/C дошта бошем, арзёбии тахминии сиркулятсияро диҳед.

4. Масири чоркунҷаи ABCD дар майдони электрикӣ муайян шудааст. Дар ҳар як буриш, \(\vec{E}\) маълум аст. Сиркулятсияро бо истифода аз формулаи скалярии \(\vec{E} \cdot d\vec{l}\) барои ҳар як буриш ҳисоб кунед ва ҷамъ кунед.

5. Фаҳмонед, ки чаро сиркулятсияи майдони электростатикӣ ҳамеша баробар ба сифр аст ва чӣ гуна он бо майдонҳои ғайриэлектростатикӣ (масалан, аз ҷониби индуксияи тағйирёбанда) фарқ мекунад.

Потенсиали электростатикӣ

Назарияи асосӣ

Потенсиали электростатикӣ — як андозаи энергияи потенсиалӣ барои як заряди воҳид дар нуқтаи додашудаи майдони электрикӣ мебошад. Он муайян мекунад, ки то чӣ андоза қувваи электрикӣ метавонад ба ҷисм кор анҷом диҳад.

Моҳият:

  • Потенсиал — бузургии скалярӣ мебошад
  • Масофа аз заряд манбаи майдон таъсир мерасонад
  • Заряд аз нуқтаи A ба B ҳаракат кунад → кори майдон = фарқи потенсиал

Формулаҳои асосӣ

V = A / q

Потенсиал — кор нисбат ба заряди санҷишӣ

V = k·(q / r)

Потенсиали тавлидшуда аз тарафи як заряд дар масофаи r

ΔV = VB - VA = -∫𝐄·d𝐥

Фарқи потенсиал байни ду нуқта

Шарҳ:

Расми мазкур мафҳуми потенсиали электростатикӣ-ро дар атрофи заряди мусбат q нишон медиҳад. Хатҳои даврашакл дар гирди заряди марказӣ — инҳо сатҳҳои изопотенсиалӣ мебошанд, ки дар онҳо потенсиал тағйир намеёбад. Бо наздик шудан ба марказ, яъне ба манбаи заряд, потенсиал афзоиш меёбад: V2 > V1.

Вектори \(\vec{r}\) масофаи байни нуқтаҳои A ва B-ро нишон медиҳад, ва \(\Delta r\) ҷойивазкунии зарядро байни ду сатҳи потенсиалӣ нишон медиҳад.

Потенсиал ба воситаи формулаи зерин муайян карда мешавад:

\[V = \frac{W}{q}\]

Ин формула нишон медиҳад, ки потенсиали электростатикӣ баробар ба кори иҷрошуда (W) барои интиқоли заряди санҷишӣ (q) аз нуқтаи A ба B мебошад.

Таҷрибаи назариявӣ

Таъҷриба: Дар як майдони электрикӣ ду нуқтаи A ва B интихоб мекунем. Заряди хурдро аз A ба B мегузаронем. Агар B дар самти майдон бошад, заряд суръат мегирад — ин маънои фарқи потенсиал дорад.

Ҳисоб: Агар A = 0.001 J ва q = 2μC бошад:

V = A/q = 0.001 / (2×10⁻⁶) = 500 V

Масъалаҳои намунавӣ

Масъала 1 (Содда): Кор барои интиқоли заряди 1μC дар майдон 0.002 J аст. Потенсиалро ёбед.

Ҳал:

V = A/q = 0.002 / (1×10⁻⁶) = 2000 V

Масъала 2 (Миёна): Заряди нуқтавӣ q = 3μC дар нуқтае потенсиал тавлид мекунад. Масофа = 0.2 м. Потенсиалро ёбед.

Ҳал:

V = k·q/r = (9×10⁹)·(3×10⁻⁶)/(0.2) = 135000 V

Масъала 3 (Мураккаб): Фарқи потенсиал байни ду нуқта 1200 V аст. Заряди -2μC аз A ба B ҳаракат мекунад. Корро ҳисоб кунед.

Ҳал:

A = q·ΔV = (-2×10⁻⁶)·1200 = -0.0024 J

Кор манфист → майдон бар зидди ҳаракат кори мусбат мекунад

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди нуқтавии +3μC дар масофаи 0.2 м ҷойгир аст. Потенсиали электростатикӣ дар ин нуқтаро ҳисоб кунед (k = 9×10⁹).

2. Ду нуқта бо потенсиалҳои 120 В ва 60 В муайян шудаанд. Агар заряд бо қимати +1μC байни онҳо ҳаракат кунад, чӣ қадар кор анҷом мешавад?

3. Агар заряд +5μC аз нуқтае бо потенсиали 200 В ба нуқтае бо потенсиали 50 В ҳаракат кунад, тағйири энергияи потенсиалии он чӣ қадар аст?

4. Дар нуқтае, ки масофа аз заряди -2μC баробар ба 0.5 м аст, потенсиалро ҳисоб кунед ва самти ҳаракати зарядҳои мусбатро шарҳ диҳед.

5. Заряди санҷишии +1μC ба фосилаи 0.3 м аз заряди +4μC ҷойгир аст. Потенсиали электростатикӣ ва энергияи потенсиалии он зарядро дар ин нуқта ҳисоб кунед.

Шадидият ҳамчун градиенти потенсиал

Назарияи асосӣ

Шадидияти майдони электрикӣ ба тағйироти потенсиал дар фазо алоқаманд аст. Ин робита бо ёрии градиент тавсиф мешавад: майдони электрикӣ ба самти коҳиши потенсиал нигаронида шудааст.

Моҳият:

  • Майдони электрикӣ векторест, ки ба самти пастшавии потенсиал ишора мекунад
  • Қимати майдон — таносуби тағйироти потенсиал ба масофа
  • Ин робита дар шакли дифференсиалӣ истифода мешавад

Формулаҳои асосӣ

𝐄 = -grad(V)

Майдони электрикӣ — градиенти манфии потенсиал

E = -ΔV / Δx

Барои ҳолати якванд (якҷаҳат)

Шарҳ:

Ин расм робитаи байни шиддати майдони электрикӣ ва градиенти потенсиал-ро нишон медиҳад. Майдон ҳамеша ба самти коҳиши потенсиал нигаронида шудааст ва бо формулаи зерин муайян карда мешавад:

\[\vec{E} = -\frac{\Delta V}{\Delta x}\]

Яъне, шиддати майдон ба тағйирёбии потенсиал вобаста аст ва самти он ба самти камшавии потенсиал ишора мекунад.

Таҷрибаи назариявӣ

Таъҷриба: Агар мо ду нуқтаи A ва B дошта бошем, ки дар онҳо потенсиалҳо VA = 500 V ва VB = 200 V, ва масофа байни онҳо 0.1 м бошад, пас мо метавонем бо ёрии қонуни E = -ΔV/Δx шадидияти майдонро чен кунем.

E = -(200 - 500) / 0.1 = 3000 V/m

Масъалаҳои намунавӣ

Масъала 1 (Содда): Потенсиал аз 100 V то 0 V дар масофаи 0.05 м коҳиш меёбад. Шадидияти майдонро ёбед.

Ҳал:

E = -ΔV/Δx = -(0 - 100)/0.05 = 2000 V/m

Масъала 2 (Миёна): Дар байни ду нуқта, потенсиал 300 V ва 150 V аст. Агар масофа 10 см бошад, шадидияти майдонро ҳисоб кунед.

Ҳал:

E = -(150 - 300)/0.1 = 1500 V/m

Масъала 3 (Мураккаб): Дар фазо функсияи потенсиал бо шакли V(x) = 500 - 20x дода шудааст. Дар нуқтаи x = 2 м шадидияти майдонро ёбед.

Ҳал:

Тавре ки V(x) = 500 - 20x → grad(V) = dV/dx = -20

E = -dV/dx = -(-20) = 20 V/m

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Агар потенсиал аз 120 В то 60 В дар масофаи 0.2 м тағйир ёбад, шадидияти майдонро ҳисоб кунед.

2. Потенсиали нуқтае дар масофаи 0.1 м аз заряд 2μC бо формулаи V = kq/r муайян карда шудааст. Шадидияти майдонро бо градиенти V ҳисоб кунед.

3. Дар хатти рост потенсиал бо суръати доимӣ коҳиш меёбад: ΔV = -90 В дар 0.3 м. Шадидият ва самти он чӣ гуна аст?

4. Агар функсияи потенсиал чунин бошад: V(x) = 100 - 5x (x — дар метр), пас E(x)-ро бо истифода аз градиент ҳисоб кунед.

5. Заряд +1μC аз нуқтае бо V = 200 В ба нуқтае бо V = 50 В ҳаракат мекунад. Масофа байни нуқтаҳо 0.5 м аст. E ва кори қувваи электрикӣ чӣ қадар аст?

Сатҳҳои эквипотенсиалӣ ва фарқи потенсиалҳо

Назарияи асосӣ

Сатҳҳои эквипотенсиалӣ — маҷмӯи нуқтаҳо дар фазо мебошанд, ки дар онҳо потенсиали электрикӣ якхела аст. Яъне, дар чунин сатҳҳо, қувваи электрикӣ кори механикӣ анҷом намедиҳад.

Хусусиятҳои асосӣ:

  • Майдони электрикӣ ҳамеша ба сатҳи эквипотенсиалӣ перпендикуляр мебошад
  • Фарқи потенсиал байни ду сатҳ = кори қувва
  • Дар дохили як сатҳ кори майдон = 0

Формулаҳои асосӣ

A = q·(VA - VB)

Кор аз нуқтаи A ба B = заряди санҷишӣ × фарқи потенсиал

ΔV = VA - VB

Фарқи потенсиал байни ду нуқта

Шарҳ:

Сатҳҳои эквипотенсиалӣ хатҳое мебошанд, ки дар онҳо потенсиали электрикӣ якхела аст. Онҳо ҳамеша ба самти шадидияти майдон перпендикуляр буда, фарқи потенсиал байни онҳо бо ΔV нишон дода мешавад. Қароргирии зичтари ин хатҳо ба шадидтари майдон ишора мекунад.

Таҷрибаи назариявӣ

Таъҷриба: Дар об як манбаи зарядро ҷойгир мекунем ва сатҳҳои потенсиалро бо сенсори потенсиал муайян мекунем. Он нуқтаҳое, ки потенсиалашон якхела аст, сатҳҳои эквипотенсиалӣ мебошанд. Ба онҳо қувваи майдон ҳамеша перпендикуляр аст.

Мисол: Агар VA = 600 V, VB = 400 V ва q = 1μC →

A = q·ΔV = 1×10⁻⁶ · (600 - 400) = 0.0002 J

Масъалаҳои намунавӣ

Масъала 1 (Содда): Ду нуқта дар фазо V = 100 V доранд. Заряде дар дохили онҳо ҳаракат мекунад. Чӣ қадар кор анҷом мешавад?

Ҳал:

Фарқи потенсиал = 0 →

A = q·ΔV = q·0 = 0

Кор = 0

Масъала 2 (Миёна): Заряди 2μC аз нуқтае бо потенсиали 500 V ба нуқтае бо 300 V ҳаракат мекунад. Корро ёбед.

Ҳал:

A = q·(VA - VB) = 2×10⁻⁶ · (500 - 300) = 0.0004 J

Масъала 3 (Мураккаб): Ду сатҳи эквипотенсиалӣ дар масофаи 0.05 м ҷойгиранд. Фарқи потенсиал байни онҳо 250 V аст. Шадидияти майдон ва кори интиқоли заряди 3μC-ро ёбед.

Ҳал:

E = ΔV / d = 250 / 0.05 = 5000 V/m
A = q·ΔV = 3×10⁻⁶ · 250 = 0.00075 J

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду нуқта дар майдон бо потенсиалҳои 100 В ва 40 В ҷойгиранд. Агар масофа байни онҳо 0.2 м бошад, шадидияти майдонро ҳисоб кунед.

2. Заряди +2μC аз нуқтаи A бо потенсиали 150 В ба нуқтаи B бо 90 В ҳаракат мекунад. Тағйири энергияи потенсиалии онро ёбед.

3. Дар расм сатҳҳои эквипотенсиалӣ бо фосилаи 1 см ва тағйири потенсиал 20 В нишон дода шудаанд. Шадидияти майдонро ҳисоб кунед.

4. Барои нуқтаҳои A ва B, ки дар як сатҳи эквипотенсиалӣ ҷойгиранд, фарқи потенсали байни онҳо чӣ қадар аст? Қувва ба заряд чӣ кор мекунад?

5. Самти шадидияти майдон нисбат ба сатҳҳои эквипотенсиалӣ чӣ гуна муайян карда мешавад ва чӣ гуна самти он ба тағйири потенсиал вобаста аст?

Поляризатсияи диэлектрикҳо

Назарияи асосӣ

Поляризатсия — равандест, ки ҳангоми таъсири майдони электрикӣ ба диэлектрикҳо рӯй медиҳад. Дар натиҷа, молекулаҳои диэлектрик ҳамоҳанг (ориентир) мешаванд ва заряди мусбат ва манфӣ аз маркази якдигар ҷудо мешаванд.

Моҳият:

  • Диэлектрикҳо моддаҳое мебошанд, ки заряди озод надоранд
  • Поляризатсия — ҷудошавии нисбии заряди дохилии молекула
  • Дар натиҷа, майдони дохилии диэлектрик коҳиш меёбад

Формулаҳои асосӣ

𝐏 = q·d

Момент ё шиддати диполӣ (q — заряд, d — фосила байни марказҳо)

𝐃 = ε₀𝐄 + 𝐏

Интенсивияти иловашудаи майдон дар дохили диэлектрик

ε = D / (ε₀E)

Эпсилон — доимии нисбии диэлектрик

Шарҳ:

Ҳангоми ҷойгир кардани диэлектрик дар майдони электрикӣ, зарядҳои дохилии он поляризатсия мешаванд ва майдони умумӣ дар дохил коҳиш меёбад. Аз ин рӯ, диэлектрикҳо қодир ба нигоҳдории энергия ва кам кардани таъсири майдон мебошанд.

Таҷрибаи назариявӣ

Таъҷриба: Агар байни ду табақаи конденсатор диэлектрик гузошта шавад, бо ёрии волтметр чен кардан мумкин аст, ки шиддати майдон кам мешавад. Ин нишон медиҳад, ки поляризатсия таъсири майдонро суст мекунад.

Мисол: Агар E = 1000 V/m ва ε = 2, пас майдони дохилӣ E' = E / ε = 500 V/m

Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар диэлектрик ε = 4 дошта бошад ва майдони беруна E = 800 V/m бошад, шадидияти майдон дар дохили модда чанд аст?

Ҳал:

E' = E / ε = 800 / 4 = 200 V/m

Масъала 2 (Миёна): Заряди 2×10⁻⁹ C ва фосилаи 0.02 м. Моменти диполиро ёбед.

Ҳал:

P = q·d = 2×10⁻⁹ · 0.02 = 4×10⁻¹¹ C·m

Масъала 3 (Мураккаб): Агар D = 4×10⁻⁸ C/m² ва E = 2×10³ V/m, бо доимии ε₀ = 8.85×10⁻¹², доимии нисбии диэлектрикро ёбед.

Ҳал:

ε = D / (ε₀E) = 4×10⁻⁸ / (8.85×10⁻¹²·2000) ≈ 2.26

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар майдони электрикӣ бо шадидияти 3000 N/C, плитаи диэлектрикӣ ҷойгир карда шудааст. Агар εr = 2.5 бошад, шадидияти майдон дар дохили диэлектрикро ҳисоб кунед.

2. Диэлектрики бо ғафсии 5 мм ва εr = 4 дар байни ду лавҳаи конденсатор гузошта шудааст. Фарқи потенсиал 100 В аст. Шадидияти майдонро дар дохили диэлектрик ёбед.

3. Тавсиф намоед, ки чӣ гуна диполҳои молекулавии диэлектрик ҳангоми таъсири майдони электрикӣ поляризатсия мешаванд.

4. Агар баъд аз поляризатсия, майдон аз 4000 N/C ба 1500 N/C коҳиш ёбад, қувваи муқобили поляризатсияро муайян кунед.

5. Барои диэлектрики муайян, шадидияти майдон E = 2000 N/C ва поляризатсия ба самти муқобил 500 N/C аст. Шадидияти умумии софро ҳисоб кунед.

Ғеҷиши электрикӣ ва теоремаи Гаусс дар диэлектрик

Назарияи асосӣ

Ғеҷиши электрикӣ (электрическая индукция) — вектори хосест, ки таъсири ҳам зарядҳои озод ва ҳам поляризатсияи дохилии диэлектрикро дар муҳити электрикӣ нишон медиҳад. Он дар баробари шиддатнокии майдон ва поляризатсия қарор дорад.

Моҳият:

  • Майдони D вобаста ба таъсири майдон ба муҳити диэлектрикӣ аст
  • Дар муҳити диэлектрикӣ флюкси майдон тавассути D ҳисоб мешавад
  • Теоремаи Гаусс барои муҳити диэлектрикӣ тағйир меёбад

Формулаҳои асосӣ

𝐃 = ε₀𝐄 + 𝐏

Вектори ғеҷиши электрикӣ — суммаи таъсири майдон ва поляризатсия

∮𝐃·d𝐒 = qозод

Теоремаи Гаусс дар муҳити диэлектрик: флюкси D = суммаи зарядҳои озод

D = ε₀·ε·E

Агар муҳит якхела ва хати бошад

Шарҳ:

Дар муҳити диэлектрик, зарядҳо метавонанд ба ду қисм ҷудо шаванд: зарядҳои озод ва зарядҳои пайваст (индукссионӣ). Танҳо зарядҳои озод ба флюкси майдон таъсир мерасонанд. Аз ҳамин сабаб, формулаи Гаусс тағйир меёбад ва бо вектори D ҳисоб мешавад.

Таҷрибаи назариявӣ

Таъҷриба: Ба конденсатор диэлектрик гузошта мешавад. Бо ёрии электрометр ё сенсор чен мешавад, ки чӣ гуна флюкси майдон бо тағйири диэлектрик дигар мешавад.

Мисол: Агар qозод = 3×10⁻⁹ C ва S = 0.01 m², пас:

D = q / S = 3×10⁻⁹ / 0.01 = 3×10⁻⁷ C/m²

Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар флюкси ғеҷишӣ D = 4×10⁻⁷ C/m² ва ε = 2 бошад, майдони электрикӣ чанд аст?

Ҳал:

E = D / (ε₀·ε) = 4×10⁻⁷ / (8.85×10⁻¹²·2) ≈ 2.26×10⁴ V/m

Масъала 2 (Миёна): Сатҳи пӯшида бо масоҳати 0.02 m² дорад ва дар дохилаш qозод = 5×10⁻⁹ C ҷойгир аст. Флюкси майдонро ҳисоб кунед.

Ҳал:

Φ = ∮𝐃·d𝐒 = qозод = 5×10⁻⁹ C
D = q / S = 5×10⁻⁹ / 0.02 = 2.5×10⁻⁷ C/m²

Масъала 3 (Мураккаб): Агар E = 3×10³ V/m ва диэлектрик бо ε = 4, ҳисоб кунед: D ва флюкс дар сатҳи 0.015 m².

Ҳал:

D = ε₀·ε·E = 8.85×10⁻¹² · 4 · 3000 = 1.062×10⁻⁷ C/m²
Φ = D · S = 1.062×10⁻⁷ · 0.015 ≈ 1.59×10⁻⁹ C

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар дохили диэлектрик шадидияти майдон E = 2000 N/C ва ε = 4ε₀ мебошад. Вектори ғеҷиши электрикиро (D) ҳисоб кунед.

2. Заряди озод дар дохили сатҳи пӯшида 6μC аст. Флюкси вектори D аз рӯи ин сатҳро ҳисоб кунед.

3. Дар як муҳити диэлектрикӣ бо εr = 3.5, шадидияти майдон 800 N/C аст. D ва P (поляризатсия)-ро ҳисоб кунед.

4. Агар \(\vec{D}\) = 1.5×10⁻⁵ C/m² бошад ва ε₀ = 8.85×10⁻¹² C²/N·m², шадидияти майдонро дар муҳити холӣ ёбед.

5. Теоремаи Гаусс-ро барои муҳити диэлектрикӣ дар ҳолати мавҷуд будани танҳо заряди озод навишта, тавзеҳ диҳед, ки чаро \(\nabla \cdot \vec{D} = \rho_{\text{озод}}\) иҷро мешавад.

Шартҳои сарҳадӣ дар муҳитҳои диэлектрикӣ

Назарияи асосӣ

Ҳангоми гузаштани майдони электрикӣ аз як муҳит ба муҳити дигар (масалан аз ҳаво ба диэлектрик), шадидият ва флюкси он метавонанд тағйир ёбанд. Шартҳои сарҳадӣ барои таҳлили ин тағйиротҳо истифода мешаванд.

Моҳият:

  • Майдони нормалӣ D (флюкс) — доим аст, агар заряд вуҷуд надошта бошад
  • Компонентаи тангенсиалии майдон E — якхела боқӣ мемонад
  • Дар мавриди сатҳи заряднок, тағйироти D вобаста ба σ аст

Формулаҳои асосӣ

D1⊥ - D2⊥ = σс

Фарқи компонентҳои нормалии D вобаста ба зичии сатҳӣ

E1∥ = E2∥

Компонентаҳои тангенсиалӣ доиманд

D = ε₀·ε·E

Ин робита дар ҳар муҳит барои ҳисоб истифода мешавад

Шарҳ:

Агар дар сарҳади байни ду муҳит заряди сатҳӣ вуҷуд надошта бошад, пас компонентҳои нормалии D ва тангенсиалии E тағйир намеёбанд. Агар бошад — тағйир ба андозаи зичии заряд (σ) сурат мегирад.

Таҷрибаи назариявӣ

Таъҷриба: Як конденсатор бо ду муҳити гуногун (масалан, ҳаво ва шиша) пур мешавад. Бо ёрии сенсори майдон ё мултиметр метавон дид, ки майдон ва потенсиал дар ҳар як муҳит чӣ гуна тағйир меёбад.

Натиҷа: E ва D дар ҳар муҳит аз ε вобастаанд. Ҳамеша D⊥ мӯътадил мемонад (агар σ = 0).

Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар D1⊥ = D2⊥ ва σ = 0, пас фарқи майдон байни муҳитҳо чӣ гуна хоҳад буд?

Ҳал:

Дар сурати σ = 0, D дар ҳар ду муҳит якхела мемонад. Аммо E тағйир меёбад вобаста ба ε:

E = D / (ε₀·ε)

Яъне E дар муҳити бо ε зиёд — хурдтар мешавад.

Масъала 2 (Миёна): Дар сарҳади ду муҳит ε1 = 1 ва ε2 = 4, ва D = 8.85×10⁻¹² C/m². Шадидияти майдонро дар ҳар муҳит ҳисоб кунед.

Ҳал:

E1 = D / (ε₀·1) = 8.85×10⁻¹² / 8.85×10⁻¹² = 1 V/m
E2 = D / (ε₀·4) = 8.85×10⁻¹² / (8.85×10⁻¹²·4) = 0.25 V/m

Масъала 3 (Мураккаб): Дар сарҳади муҳитҳо бо зичии сатҳии заряд σ = 5×10⁻⁸ C/m², ҳисоб кунед фарқи D⊥ дар ду тарафи сарҳад.

Ҳал:

D1⊥ - D2⊥ = σ = 5×10⁻⁸ C/m²

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду муҳити диэлектрикӣ бо ε₁ = 2ε₀ ва ε₂ = 5ε₀ ҳамсарҳад мебошанд. Агар E₁ = 3000 N/C бошад, арзиши E₂-ро дар самти нормалӣ муайян кунед.

2. Вектори шадидияти майдон дар як муҳит перпендикуляр ба сарҳад ворид мешавад. Агар D₁ = D₂ бошад, чӣ хулоса метавон баровард?

3. Агар майдон E ба самти таносубан параллел ба сарҳад ворид шавад, чӣ гуна тағйири он дар муҳити дигар ба амал меояд? Бо ёрии ε₁ ва ε₂ шарҳ диҳед.

4. Дар нуқтаи сарҳад байни ду диэлектрик, чӣ шартҳо барои пайвастагии Etangential ва Dnormal иҷро мешаванд? Бо формула шарҳ диҳед.

5. Агар дар муҳити аввал D₁⊥ = 4×10⁻⁵ C/m² ва ε₁ = 2ε₀ бошад, шадидияти майдонро дар муҳити аввал ва дуюм (ε₂ = 6ε₀) ҳисоб кунед.

Сегнетоэлектрикҳо ва хосиятҳои онҳо

Назарияи асосӣ

Сегнетоэлектрикҳо — ин намуди махсуси диэлектрикҳо мебошанд, ки қодир ба нигоҳдории поляризатсияи доимӣ дар ҳолати набудани майдони беруна мебошанд. Онҳо ба монанди магнитҳои электрикӣ мебошанд.

Хусусиятҳои асосӣ:

  • Дорои поляризатсияи худӣ дар ҳолати бе майдон
  • Ҳангоми баръакс кардани майдон — поляризатсия низ тағйир меёбад
  • Нишон медиҳанд гистерезис дар графикаи "P–E" (поляризатсия-шадидият)
  • Эффекти қатъии ҳароратӣ (пойнти Кюри)

Формулаҳои асосӣ ва графикҳо

Сегнетоэлектрикҳо тавассути графикаи гистерезис тавсиф мешаванд:

Pостаточный ≠ 0, вақте E = 0
P(E) — контури гистерезис

Дар ҳарорати баландтар аз нуқтаи Кюри — сегнетоэлектрик хосияти худро гум мекунад

Шарҳ:

Сегнетоэлектрикҳо ба мисли BaTiO₃ ё KH₂PO₄ дар электроникаи муосир истифода мешаванд. Онҳо ба таъсири шадиди майдон хеле ҳассосанд ва метавонанд дар ҳолати гуногун нигоҳ доранд — ки барои хотираҳо (RAM, EEPROM) бисёр муҳим аст.

Таҷрибаи назариявӣ

Таъҷриба: Дар як асбоби мувозинатӣ сегнетоэлектрикро ҷойгир мекунем ва шиддатнокии майдонро тағйир медиҳем. Мо мебинем, ки поляризатсия дар даври аввал зиёд шуда, вале ҳангоми баргардонидани майдон боқӣ мемонад. Ин боиси гистерезис мегардад.

Натиҷа: Диаграммаи P(E) контури гистерезисро нишон медиҳад.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Чаро сегнетоэлектрик метавонад поляризатсияро нигоҳ дорад ҳатто баъд аз хомӯш кардани майдон?

Ҷавоб:

Ба сабаби сохтори дохилии асимметрӣ ва қобилияти нигоҳ доштани танзими диполҳо. Ин хосият "поляризатсияи боқимонда" номида мешавад.

Масъала 2 (Миёна): Агар графикаи гистерезис нишон диҳад, ки Pостаточный = 0.1 C/m² ва Eкоҳиш = 2×10⁵ V/m, чӣ арзиши майдон лозим аст барои нест кардани поляризатсия?

Ҷавоб:

Майдони E ≈ -Eкоҳиш = -2×10⁵ V/m лозим аст, то поляризатсия ба 0 расад.

Масъала 3 (Мураккаб): Сегнетоэлектрик дар ҳарорати T < TКюри бо ε = 1000 ҷойгир аст. Пас аз гузариш аз нуқтаи Кюри, ε ба 10 паст мешавад. Чаро?

Шарҳ:

Дар ҳарорати T > TКюри, мавод хосияти сегнетоэлектрикӣ гум мекунад ва монанди диэлектрики оддӣ рафтор мекунад. Аз ин сабаб, қобилияти поляризатсия коҳиш ёфта, ε хурд мешавад.

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Тавсиф диҳед, ки чӣ гуна сегнетоэлектрикҳо бо майдони беруна поляризатсия мешаванд ва чӣ гуна самти диполҳоро тағйир медиҳанд.

2. Дар натиҷаи таъсири майдон, як кристали сегнетоэлектрикӣ поляризатсия пайдо мекунад. Агар қимати \(\vec{P}\) = 6×10⁻⁶ C/m² бошад ва сатҳи он 0.02 м², зарядҳои рӯизаминиро ҳисоб кунед.

3. Дар сегнетоэлектрикӣ, дар ҳарорати баландтар аз ҳарорати Кюри, чӣ тағйирот дар хосиятҳои электрикӣ ба амал меояд? Бо мисол шарҳ диҳед.

4. Чаро хати гистерезиси сегнетоэлектрикҳо шакли ҳалқавӣ дорад? Сохтори онро тасвир кунед ва бо поляризатсияи боқимонда шарҳ диҳед.

5. Агар дар кристал поляризатсияи боқимонда 3×10⁻⁶ C/m² боқӣ монад, чӣ гуна ин ба кори таҷҳизот (масалан, сенсор ё хотираи барқӣ) таъсир мерасонад?

Нақши ноқилҳо дар майдони электростатикӣ

Назарияи асосӣ

Дар муҳити электростатикӣ, ноқилҳо (металлҳо) хосиятҳои махсус доранд, зеро электронҳо метавонанд озодона ҳаракат кунанд. Дар натиҷаи таъсири майдон, заряди озод дар дохили ноқил ҷобаҷо шуда, ноқил ба ҳолати тавозуни электростатикӣ мерасад.

Хусусиятҳои асосӣ:

  • Майдони электрикӣ дар дохили ноқили мутавозин = 0
  • Зарядҳо танҳо дар сафҳаи беруна ҷамъ мешаванд
  • Ноқилҳо экранизатсия мекунанд — онҳо майдони дохилӣ надоранд
  • Сатҳи ноқил — сатҳи эквипотенсиалӣ аст

Принсипҳои асосӣ

Ein = 0

Майдони электрикӣ дар дохили ноқил = 0

qсафҳа = Σq

Ҳамаи зарядҳо дар рӯйи берунӣ ҷамъ мешаванд

V = const

Ҳама нуқтаҳои ноқил як потенсиал доранд

Шарҳ:

Ноқилҳо дар системаҳои электростатикӣ ҳамчун экрани электрикӣ истифода мешаванд. Заряд дар берун аз ноқил дар атрофи он майдон эҷод мекунад, вале дохили он комилан муҳофизатшуда мемонад. Ин хосият барои муҳофизати дастгоҳҳо аз майдонҳо муҳим аст.

Таҷрибаи назариявӣ

Таъҷриба: Як тӯби металлӣ пур аз зарядро дар муҳити вакуум ҷойгир мекунем ва бо электрометр дар дохили он майдонро месанҷем. Мебинем, ки E = 0 аст. Аммо дар беруни тӯб — майдон вуҷуд дорад.

Ин натиҷа экранизацию тасдиқ мекунад.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Чаро майдони электрикӣ дар дохили тӯби металлии заряднок баробар ба сифр аст?

Шарҳ:

Зарядҳо дар сатҳи берунӣ тақсим мешаванд. Ҷамъбасти вектории таъсири ҳамаи элементҳои заряди рӯй дар дохили ноқил бекор мешаванд → E = 0

Масъала 2 (Миёна): Заряди 5μC ба рӯйи як ноқили сферӣ гузошта шудааст. Масофа аз марказ 10 см. Майдони электрикӣ дар дохили сфераро ёбед.

Ҳал:

E = 0 (дар дохили ноқили мутавозин)

Масъала 3 (Мураккаб): Нуқтаи бо заряди -2μC дар назди рӯйи як ноқили сатҳи заряднок ҷойгир аст. Бо чӣ шарт ноқил майдони дохилиашро нигоҳ медорад?

Ҷавоб:

Ноқил боиси индуксияи зарядҳои муқобил дар рӯяш мегардад, ки майдонро дар дохили худ комилан бекор мекунад. Шарт: ноқил бояд комилан пӯшида ва мутавозин бошад.

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар майдони беруна як тӯби ноқил бо радиуси R ҷойгир аст. Дар дохили он чӣ гуна шадидияти майдон тағйир меёбад? Шарҳ диҳед.

2. Зарядҳои озод дар дохили ноқили мутавозин чӣ гуна тақсим мешаванд ва чаро сатҳи ноқил эквипотенсиалӣ ҳисобида мешавад?

3. Агар заряд Q = +4μC ба сатҳи ноқил гузошта шавад, зичии сатҳӣ дар нуқтае бо масоҳати 1 см² чӣ қадар аст?

4. Вақте ки ноқили холӣ ба майдони барқӣ ворид мешавад, чӣ гуна дар сатҳи он заряди индуксионӣ ба вуҷуд меояд? Намунаи графикӣ пешниҳод кунед.

5. Чаро дар дохили қуттии ноқил (экрани электростатикӣ) шадидияти майдон ба сифр мерасад? Ба сифати мисол, камераи Фарадейро шарҳ диҳед.

Конденсатор ва ғунҷоиши он

Назарияи асосӣ

Конденсатор — дастгоҳест, ки қодир аст заряд ва энергияи электростатикӣ нигоҳ дорад. Он одатан аз ду ноқили ҷудошуда бо диэлектрик иборат аст.

Хусусиятҳои асосӣ:

  • Ду табақи металлӣ бо масофаи муайян ва диэлектрик дар миён
  • Бо потенсиали гуногун пур мешаванд
  • Заряд дар рӯи табақҳо ҷамъ мешавад
  • Энергия дар майдони электрикӣ нигоҳ дошта мешавад

Формулаҳои асосӣ

C = q / U

Ғунҷоиши (ёмкость) конденсатор: таносуби заряд ба фарқи потенсиал

C = ε₀·ε·S / d

Барои конденсатори табақавӣ: вобастагӣ аз масоҳат, масофа ва муҳит

W = (1/2)·C·U²

Энергияи захиравии конденсатор

Шарҳ:

Ғунҷоиши конденсатор вобаста ба геометрия ва муҳити байни табақҳост. Диэлектрик ғунҷоишро зиёд мекунад ва майдонро коҳиш медиҳад. Конденсаторҳо дар схемаҳо барои нигоҳдорӣ, филтркунӣ ва генераторҳо истифода мешаванд.

Таҷрибаи назариявӣ

Таъҷриба: Бо кӯмаки электрометр, ду табақи металлӣ пур карда мешаванд. Бо зиёд кардани фосила d ё диэлектрик, тағйироти шиддат ва ғунҷоиш мушоҳида мешавад. Ғунҷоиш кам ё зиёд мешавад.

Натиҷа: C ~ 1/d, C ~ ε, C ~ S

Масъалаҳои намунавӣ

Масъала 1 (Содда): Конденсатор дорои 10μC заряд ва 100 V шиддат аст. Ғунҷоишро ёбед.

Ҳал:

C = q / U = 10×10⁻⁶ / 100 = 1×10⁻⁷ F = 100 nF

Масъала 2 (Миёна): Масоҳати табақҳо S = 0.01 m², масофа d = 2 mm, ε = 5. Ғунҷоиши конденсаторро ёбед.

Ҳал:

C = ε₀·ε·S / d = 8.85×10⁻¹² · 5 · 0.01 / 0.002 = 2.2×10⁻¹⁰ F = 220 pF

Масъала 3 (Мураккаб): Агар конденсатор бо C = 200 nF ба 50 V пайваст шавад, чӣ қадар энергия захира мешавад? Агар ε ду баробар зиёд шавад, чӣ мешавад?

Ҳал:

W = (1/2)·C·U² = 0.5 · 200×10⁻⁹ · 2500 = 2.5×10⁻⁴ J

Агар ε ду баробар зиёд шавад → C низ ду баробар зиёд → энергия W = 5×10⁻⁴ J

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду плитаи параллелӣ бо масоҳати 100 см² ва фосилаи 2 мм бо ҳаво пур шудаанд. Ғунҷоиши конденсаторро ҳисоб кунед.

2. Агар конденсатор бо ғунҷоиши 10 μF ба фарқи потенсиали 12 В пайваст бошад, чӣ миқдор заряд дар он захира мешавад?

3. Конденсаторе бо диэлектрики εr = 4 ва ғунҷоиши 5 μF дорем. Агар диэлектрик хориҷ карда шавад, ғунҷоиши нав чӣ қадар мешавад?

4. Барои конденсаторе бо C = 6 μF ва V = 10 В, энергияи захирашуда дар майдони он чӣ қадар аст?

5. Агар конденсаторро пур карда, сипас аз манбаъ ҷудо намоем ва диэлектрик ворид кунем, чӣ гуна заряди он, ғунҷоиш ва потенсиал тағйир меёбанд? Шарҳ диҳед.

Энергияи майдони электростатикӣ

Назарияи асосӣ

Майдони электростатикӣ метавонад энергия нигоҳ дорад. Ин энергия ҳангоми ҷудо кардани зарядҳо ва таъсири электрикӣ дар муҳит тавлид мешавад. Дар системаҳои электрикӣ, мисли конденсатор, ин энергия дар фазои байни табақҳо ҷойгир аст.

Моҳият:

  • Энергия ба воситаи шиддати майдон ва ғунҷоиш вобаста аст
  • Энергия метавонад ҳамчун функсияи заряд ё шиддат ифода шавад
  • Энергия дар фазо тақсим мешавад ва метавонад бо зичии энергия чен карда шавад

Формулаҳои асосӣ

W = (1/2)·C·U²

Энергияи захирашуда дар конденсатор

W = (1/2)·q·U

Альтернативӣ тавассути заряд ва шиддат

w = (1/2)·ε·ε₀·E²

Зичии энергия (энергия дар як воҳиди ҳаҷм)

Шарҳ:

Энергияи электростатикӣ метавонад ҳамчун манбаи қувва истифода шавад. Масалан, дар флеши камера ё тири электростатикӣ. Ҳангоми аз система баровардани энергия, он ба кор ё гармӣ табдил меёбад.

Таҷрибаи назариявӣ

Таъҷриба: Конденсаторро бо манбаи шиддат пур мекунем ва баъд онро ба лампочка ё резистор васл мекунем. Лампочка фурӯзон мешавад — нишон медиҳад, ки энергия захира шудааст.

Ҳисоб: Агар C = 1μF ва U = 100 V → W = (1/2)·10⁻⁶·100² = 0.005 J

Масъалаҳои намунавӣ

Масъала 1 (Содда): Конденсатор бо C = 500 nF ба 20 V пайваст шудааст. Энергияро ҳисоб кунед.

Ҳал:

W = (1/2)·C·U² = 0.5·500×10⁻⁹·400 = 1×10⁻⁴ J = 0.1 mJ

Масъала 2 (Миёна): Шиддати майдон E = 2×10⁴ V/m, ε = 4. Зичии энергияро ҳисоб кунед.

Ҳал:

w = (1/2)·ε·ε₀·E² = 0.5·4·8.85×10⁻¹²·(2×10⁴)² ≈ 7.08×10⁻³ J/m³

Масъала 3 (Мураккаб): Конденсатор бо масоҳати табақҳо S = 0.02 m² ва фосилаи 1 mm, бо ε = 3 пур шудааст. Агар шиддат 200 V бошад, энергияро ҳисоб кунед.

Ҳал:

Аввал ғунҷоиш:

C = ε₀·ε·S/d = 8.85×10⁻¹²·3·0.02 / 0.001 = 5.31×10⁻¹⁰ F

Сипас энергия:

W = 0.5·C·U² = 0.5·5.31×10⁻¹⁰·(200)² ≈ 1.06×10⁻⁵ J

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Конденсаторе бо ғунҷоиши 8 μF ба шиддати 12 В пайваст шудааст. Энергияи захирашудаи онро ҳисоб кунед.

2. Агар шадидияти майдон E = 3000 N/C ва ҳаҷми фазо 5×10⁻⁴ m³ бошад, энергияи майдонро дар он ҳисоб кунед (дар муҳити ҳавоӣ).

3. Конденсаторро бо V = 100 В пур мекунем ва сипас онро ҷудо намуда, диэлектрикро ворид мекунем. Энергия чӣ гуна тағйир меёбад?

4. Зарядҳои +2μC ва -2μC дар фосилаи 0.1 м ҷойгиранд. Энергияи системаи онҳо дар вакуумро ҳисоб кунед.

5. Шарҳ диҳед: чаро майдони электростатикӣ энергияи фазоро пур мекунад ва он чӣ гуна дар системаи зарядҳо нигоҳ дошта мешавад?

Ҷараён, зичӣ ва Қонуни умумии ҷараён (ҚЭҲ)

Назарияи асосӣ

Ҷараёни электрикӣ — ҳаракати муташаккили зарядҳо дар муҳити ноқил мебошад. Он метавонад аз электронҳо, ионҳо ё сӯрохҳо ташкил ёбад. Дар ҳар як нуқтаи муҳит мо метавонем шиддат ва самти ҷараёнро тавсиф кунем.

Моҳият:

  • Қувваи ҷараён I = миқдори заряд, ки аз нуқтаи муайян дар як вақт мегузарад
  • Зичии ҷараён J = I/S — ҷараён дар як воҳиди масоҳат
  • ҚЭҲ — тағйироти зичии заряд бо ворид/баромадани ҷараён вобаста аст

Формулаҳои асосӣ

I = dq / dt

Ҷараён — заряди гузашта дар як вақт

𝐉 = I / S = nqv

Зичии ҷараён: вобаста ба зичии заряд ва суръати ҳаракат

∇·𝐉 + ∂ρ/∂t = 0

Қонуни умумии нигоҳдории заряд (ҚЭҲ)

Шарҳ:

ҚЭҲ маънои онро дорад, ки агар дар нуқтае заряд кам шавад, пас ҷараён бояд ба он ворид шуда бошад. Ҳеҷ гоҳ заряд "гум" намешавад. Ин қонун универсалӣ ва асосии физикаи электрикӣ мебошад.

Таҷрибаи назариявӣ

Таъҷриба: Ба як ноқили тӯлонӣ манбаи шиддат пайваст мешавад. Бо ёрии амперметр ҷараён дар нуқтаҳои гуногун чен мешавад. Агар ҷараён доимӣ бошад, нишон медиҳад, ки заряд нигоҳ дошта мешавад (I ворид = I хориҷ).

Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар 3 C заряд дар 2 сония гузарад, ҷараён чанд аст?

Ҳал:

I = q / t = 3 / 2 = 1.5 A

Масъала 2 (Миёна): Ҷараёни 2 A тавассути ноқили бо масоҳати 0.5 мм² мегузарад. Зичии ҷараёнро ёбед.

Ҳал:

S = 0.5 мм² = 0.5×10⁻⁶ м²
J = I / S = 2 / (0.5×10⁻⁶) = 4×10⁶ A/m²

Масъала 3 (Мураккаб): Агар ∂ρ/∂t = -5×10⁻³ C/m³·s ва ∇·𝐉 = ? Қонуни ҚЭҲ-ро истифода бурда, арзиши ∇·𝐉-ро ёбед.

Ҳал:

∇·𝐉 = -∂ρ/∂t = -(-5×10⁻³) = 5×10⁻³ A/m³

Вазифаи хонагӣ

Масъалаҳо барои ҳал

Қонуни Ом барои қитъаҳои якҷинса ва ғайриякҷинса

Назарияи асосӣ

Қонуни Ом робитаи байни шиддат (U), ҷараён (I) ва муқовимат (R)-ро тавзеҳ медиҳад. Он дар ду ҳолати асосӣ татбиқ мешавад: барои қитъаи якҷинса ва барои қитъаи ғайриякҷинса (бо манбаи ЭҚ).

Навъи қитъаҳо:

  • Қитъаи якҷинса: танҳо ноқилҳо, бе манбаъ
  • Қитъаи ғайриякҷинса: дорои манбаи ЭҚ (батарея, генератор)

Формулаҳои асосӣ

I = U / R

Қонуни Ом барои қитъаи якҷинса

I = (𝜀 - U) / r

Қонуни Ом барои қитъаи ғайриякҷинса бо ЭҚ 𝜀 ва муқовимати дохилӣ r

U = IR

Шиддат дар ноқили берунии қитъаи якҷинса

Шарҳ:

Дар қитъаи ғайриякҷинса, манбаи ЭҚ қувваи дохилиро низ илова мекунад. Муқовимати дохилии манбаъ боиси кам шудани шиддати воқеии ба ноқил дода мешавад. Қонуни Ом ҳам барои таҳлили схемаҳо ва ҳам таҷҳизоти амалӣ муҳим аст.

Таҷрибаи назариявӣ

Таъҷриба: Бо ёрии амперметр ва волтметр қитъаи ноқилро месанҷем. Дар ҳолати пайвасти мустақим бо манбаъ, зери таъсири шиддат ҷараён ба назар мерасад. Агар муқовимат зиёд шавад, ҷараён кам мешавад (I ~ 1/R).

Масъалаҳои намунавӣ

Масъала 1 (Содда): Дар ноқили бо R = 5 Ом ва шиддати 10 В ҷараён чанд аст?

Ҳал:

I = U / R = 10 / 5 = 2 A

Масъала 2 (Миёна): Манбаи ЭҚ бо 𝜀 = 12 В ва муқовимати дохилӣ r = 1 Ом ба ноқили бо R = 5 Ом пайваст шудааст. Ҷараёнро ҳисоб кунед.

Ҳал:

Муқовимати умумӣ: Rобщ = R + r = 6 Ом

I = 𝜀 / (R + r) = 12 / 6 = 2 A

Масъала 3 (Мураккаб): Манбаъи ЭҚ бо 𝜀 = 9 В, r = 0.5 Ом ба реостат бо муқовимати тағйирёбанда пайваст аст. Агар ҷараён 1.5 А бошад, чӣ қадар энергия ба реостат дода мешавад дар 2 сония?

Ҳал:

U = IR = 𝜀 - Ir = 9 - 1.5·0.5 = 8.25 В

W = U·I·t = 8.25·1.5·2 = 24.75 J

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар ноқиле бо муқовимати 5 Ом шиддати 10 В татбиқ шудааст. Ҷараёни гузарандаро ҳисоб кунед.

2. Дар схемаи ғайриякҷинса ЭҚН = 12 В, r = 1 Ом ва R = 5 Ом. Ҷараёнро дар қитъа ҳисоб кунед.

3. Ноқиле бо муқовимати 2 Ом аз батареяи 9 В ғизо мегирад. Агар ҷараён 4 А бошад, муқовимати дохилии манбаъро ёбед.

4. Барои ноқили ғайриякҷинса, агар ЭҚН = 6 В, r = 0.5 Ом ва U = 4 В бошад, ҷараёни системаро ҳисоб кунед.

5. Тафсир диҳед, ки фарқи байни ҷараёни якҷинса ва ғайриякҷинса дар чист, ва чаро ЭҚН метавонад бар шиддат ғолиб ё муқобил бошад.

Қонуни Ҷоул–Ленц

Назарияи асосӣ

Қонуни Ҷоул–Ленц мегӯяд, ки ҳангоми гузаштани ҷараёни электрикӣ аз ноқил, як қисми энергия ба шакли гармӣ табдил меёбад. Ин қонун бар асоси таъсири гармкунӣ дар муҳити муқовиматнок асос ёфтааст.

Моҳият:

  • Гармӣ ҳосил мешавад аз сабаби муқовимати ноқил
  • Миқдори гармӣ аз I², R ва t вобаста аст
  • Ин қонун дар асоси қонуни Ом ва нигоҳдории энергия баромадааст

Формулаҳои асосӣ

Q = I²·R·t

Гармиеро, ки ноқил ҷудо мекунад

Q = U·I·t

Агар шиддат маълум бошад

Q = U²·t / R

Ифода тавассути шиддат ва муқовимат

Шарҳ:

Қонуни Ҷоул–Ленц нишон медиҳад, ки бештар ҷараён ё муқовимат → бештар гармӣ. Аз ин рӯ, ноқилҳои борик ё маводҳои муқовиматнок бештар гарм мешаванд. Ин таъсир дар лампаҳо, печҳо, тавакҳо истифода мешавад.

Таҷрибаи назариявӣ

Таъҷриба: Ба симе ҷараёни доимӣ дода мешавад. Пас аз 2–3 дақиқа, ҳарорати сим афзоиш меёбад. Агар ҷараён зиёд шавад, гармӣ бештар мешавад. Ин бо формулаи Q = I²Rt мувофиқат мекунад.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Ҷараёни 2 A аз ноқили бо R = 5 Ом дар тӯли 10 сония мегузарад. Чанд джоул гармӣ ҳосил мешавад?

Ҳал:

Q = I²·R·t = 2²·5·10 = 4·5·10 = 200 J

Масъала 2 (Миёна): Ноқил бо R = 10 Ом ба манбаи 60 В пайваст аст. Дар 30 сония чӣ қадар гармӣ ҷудо мешавад?

Ҳал:

Q = U²·t / R = 60²·30 / 10 = 3600·3 = 10800 J

Масъала 3 (Мураккаб): Симе бо R = 1.5 Ом дар давоми 5 дақиқа 900 J гармӣ ҷудо мекунад. Ҷараёнро ҳисоб кунед.

Ҳал:

Q = I²·R·t → I² = Q / (R·t) = 900 / (1.5·300) = 2 → I = √2 ≈ 1.41 A

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ноқиле бо муқовимати 3 Ом бо ҷараёни 2 А дар давоми 5 дақиқа ҷорӣ мешавад. Миқдори гармии ҳосилшударо ҳисоб кунед.

2. Агар дар як қитъа U = 12 В ва R = 4 Ом бошад, дар 2 дақиқа чӣ қадар гармӣ ҳосил мешавад?

3. Ҷараёни 0.5 А дар муқовимати 10 Ом дар 1 сония мегузарад. Q-ро бо ду формулаи гуногун ҳисоб кунед ва натиҷаҳоро муқоиса намоед.

4. Агар қувваи гармии ноқил дар як сония 36 Ҷ бошад ва R = 2 Ом, ҷараёнро дар ноқил ёбед.

5. Тавзеҳ диҳед: чаро ноқил бо муқовимати баланд бештар гарм мешавад ва чӣ гуна қонуни Ҷоул–Ленц асоси кори таҷҳизоти гармидиҳӣ аст (мисли обгармкунак, печи барқӣ).

Қоидаҳои Кирхгофф

Назарияи асосӣ

Қоидаҳои Кирхгофф ду қонунро дарбар мегиранд, ки барои таҳлили схемаҳои мураккаби барқӣ истифода мешаванд:

1. Қоидаи якум (Қонун барои гиреҳҳо):

Суммаи ҷараёнҳое, ки ба гиреҳ ворид мешаванд, бо суммаи ҷараёнҳое, ки мебароранд, баробар аст.

∑Iв = ∑Iх

2. Қоидаи дуюм (Қонун барои контурҳо):

Дар ҳар як контури пӯшида суммаи шиддатҳо (бо аломатҳояшон) баробар ба сифр аст.

∑U = 0

Формулаҳои асосӣ

∑Iin - ∑Iout = 0

Барои гиреҳ (қоидаи 1)

∑(𝜀 - IR) = 0

Барои контур (қоидаи 2)

Шарҳ:

Қоидаҳои Кирхгофф ба қонунҳои нигоҳдории заряд ва энергия асос ёфтаанд. Онҳо дар схемаҳое, ки бо якчанд манбаъ ва муқовимат таҳия шудаанд, истифода мешаванд. Барои ҳисоб, одатан, системаҳои муодила ташкил медиҳанд.

Таҷрибаи назариявӣ

Таъҷриба: Схемаи дорои ду манбаи ЭҚ ва се муқовиматро месозем. Бо ёрии амперметр ҷараёнҳоро дар ҳар қитъа чен мекунем. Сипас ҳисобҳои Кирхгоффро татбиқ карда, дурустии натиҷаҳоро муқоиса мекунем.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Дар як гиреҳ 3 ҷараён: I₁ = 2 A ворид мешавад, I₂ = 1.5 A мебарояд, I₃ номаълум аст. I₃-ро ёбед.

Ҳал:

∑Iв = ∑Iх → I₁ = I₂ + I₃ → I₃ = I₁ - I₂ = 2 - 1.5 = 0.5 A

Масъала 2 (Миёна): Дар контур 𝜀 = 12 В, R₁ = 2 Ом, R₂ = 4 Ом, ва танҳо як ҷараён I мегузарад. Шиддатро дар ҳар қитъа ва I-ро ёбед.

Ҳал:

∑U = 0 → 𝜀 - IR₁ - IR₂ = 0 → I = 𝜀 / (R₁ + R₂) = 12 / 6 = 2 A

U₁ = IR₁ = 2·2 = 4 В, U₂ = 2·4 = 8 В

Масъала 3 (Мураккаб): Дар схемае ду гиреҳ ва ду контур мавҷуданд. Шумо бояд барои I₁ ва I₂ системаҳои муодила нависед ва ҳал кунед.

Роҳнамо:

  • Барои ҳар гиреҳ: ∑Iв = ∑Iх
  • Барои ҳар контур: ∑(𝜀 - IR) = 0

Бо таҳлили графикӣ ё матритсавӣ система ҳал мешавад (дарси алоҳида талаб мекунад).

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар як гиреҳ се ҷараён ворид мешаванд: I₁ = 2 A, I₂ = 3 A ва I₃ баромада меравад. Арзиши I₃-ро ҳисоб кунед.

2. Барои даври замкшуда: ЭҚН = 12 В, R₁ = 2 Ом, R₂ = 4 Ом. Ҷараёнро бо истифода аз қоидаи дуюми Кирхгофф ёбед.

3. Дар гиреҳ чор ҷараён иштирок доранд: I₁ = 1.5 A (даромада), I₂ = 0.5 A (баромада), I₃ = 2 A (даромада), I₄ = ? Арзиши I₄-ро ҳисоб кунед ва самташро муайян кунед.

4. Дар давр ду манбаи ЭҚН ва се муқовимат мавҷуданд: \(\mathcal{E}_1 = 6 В, \mathcal{E}_2 = 4 В, R_1 = 1 Ом, R_2 = 2 Ом, R_3 = 3 Ом\). Ҷараёнро бо тартиб ҳисоб кунед.

5. Шарҳ диҳед: чӣ гуна метавон бо истифода аз қоидаҳои Кирхгофф ҷараён ва шиддатро дар шабакаҳои мураккаб ёфт ва чӣ гуна системаҳои муодила сохта мешаванд.

Электрогузаронии металлҳо (назарияи классикӣ)

Назарияи асосӣ

Мувофиқи назарияи классикӣ (модели Друде), атомҳои металлҳо дорои электронҳои озод мебошанд, ки метавонанд дар дохили рештаи кристалл озодона ҳаракат кунанд. Ҳангоми таъсири майдони электрикӣ, ин электронҳо ҷараёни доимии электрикӣ ба вуҷуд меоранд.

Моҳиятҳои асосӣ:

  • Электронҳо — зарраҳои озоди ҳаракаткунанда дар рештаи кристалл
  • Майдони электрикӣ ба онҳо самт медиҳад → ҷараён
  • Ҳар як электрон ба зарраҳои решта бармехӯрад → муқовимат
  • Муқовимат вобаста ба ҳарорат зиёд мешавад

Формулаҳои асосӣ

J = nqv

Зичии ҷараён — вобаста ба зичии электронҳо (n), заряд (q) ва суръати миёна (v)

σ = nq²τ / m

Қобилияти гузаронандагӣ — тавассути вақти озод τ ва массаи электрон

R = ρ·l / S

Муқовимати ноқил бо дарозӣ l ва қиёс S

Шарҳ:

Модели классикӣ ба ҳаракати электрони озод монанд ба молекулаи гази идеалӣ нигоҳ мекунад. Гарчанде ки он хосиятҳои мураккабро пурра тавзеҳ дода наметавонад, вале барои тасвири муқовимат ва таъсири ҳарорат хело муфид аст.

Таҷрибаи назариявӣ

Таъҷриба: Ноқили мисро гарм мекунем ва ҷараёнро бо амперметр чен мекунем. Ҳангоми афзоиши ҳарорат, ҷараён коҳиш меёбад. Ин аз сабаби зиёд шудани бархӯрдҳо (μ кам мешавад) мувофиқи назарияи Друде мебошад.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Зичии ҷараён дар симе 2×10⁵ A/m² аст. Агар n = 8.5×10²⁸ м⁻³, q = 1.6×10⁻¹⁹ Кл, суръати v-ро ёбед.

Ҳал:

J = nqv → v = J / (nq) = 2×10⁵ / (8.5×10²⁸ · 1.6×10⁻¹⁹) ≈ 1.47×10⁻³ м/с

Масъала 2 (Миёна): Агар τ = 2×10⁻¹⁴ с, n = 10²⁹ м⁻³, m = 9.1×10⁻³¹ кг, қобилияти гузаронандагиро (σ) ҳисоб кунед.

Ҳал:

σ = nq²τ / m = 10²⁹ · (1.6×10⁻¹⁹)² · 2×10⁻¹⁴ / 9.1×10⁻³¹ ≈ 5.6×10⁷ 1/Ом·м

Масъала 3 (Мураккаб): Ноқили мисии бо дарозии 1 м ва сатҳи 0.5 мм² дорои муқовимати 0.03 Ом аст. Муқовимати ҷисмро тавассути ρ ҳисоб кунед.

Ҳал:

ρ = R·S / l = 0.03·0.5×10⁻⁶ / 1 = 1.5×10⁻⁸ Ом·м

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар ноқили мисӣ, зичии ҷараён j = 3×10⁶ A/m² ва зичии зарядҳои озод n = 8.5×10²⁸ 1/m³ аст. Суръати ҳаракати зарядҳоро (v) ҳисоб кунед.

2. Агар қувваи майдон E = 10⁻² V/m ва σ = 5.8×10⁷ S/m (барои мис), ҷараёни зичиро ҳисоб кунед.

3. Бо истифода аз формулаи \(\sigma = \frac{n e^2 \tau}{m}\), барои мис ҳисоб кунед: агар n = 8.5×10²⁸, τ = 2.5×10⁻¹⁴ с, массаи электрон m = 9.1×10⁻³¹ кг ва e = 1.6×10⁻¹⁹ Кл.

4. Тавзеҳ диҳед: чаро дар модели классикӣ суръати зарраҳо миёнагирӣ шуда, ҷараён дар самти муайян пайдо мешавад?

5. Барои ноқил бо сатҳи буриш 1 мм² ва ҷараёни 3 А, зичии ҷараён ва суръати дрейфро ҳисоб кунед (n = 10²⁹, e = 1.6×10⁻¹⁹ Кл).

Фотоэффект ва эмиссияи термоэлектронӣ

Назарияи асосӣ

Фотоэффект — раванде, ки дар он электронҳо аз рӯи металл бар асари таъсири нури монохроматикӣ хориҷ мешаванд.

Эмиссияи термоэлектронӣ — озодшавии электронҳо аз сатҳи металл дар натиҷаи гармкунии шадиди он.

Фарқиятҳо ва умумиятҳо:

  • Ҳар ду раванд бо озод кардани электронҳо аз модда алоқаманданд
  • Фотоэффект — таъсири энергияи фотон
  • Термоэмиссия — таъсири энергияи ҳароратӣ
  • Ҳарду дар дастгоҳҳои электронии ҳассос истифода мешаванд

Формулаҳои асосӣ

Eфотон = h·ν = Aчих + Eкин

Барои фотоэффект — энергия барои чидани электрон ва ҳаракаташ меравад

j = A·T²·e−φ/kT

Барои термоэмиссия — формулаи Ричардсон

Aчих — кори чидан, h — доимии Планк, ν — басомад

Шарҳ:

Фотоэффект асоси физикаи квантӣ аст — он исбот кард, ки нур қисматчаҳо (фотонҳо) дорад. Эмиссияи термоэлектронӣ бошад, дар лампаҳои электровакуумӣ ва диодҳо васеъ истифода мешавад.

Таҷрибаи назариявӣ

Фотоэффект: Нури кабуд ба рӯи металл мебарояд → бо волтметр ҷараёни хориҷшавии электронҳоро сабт мекунем. Агар энергия аз Aчих зиёд бошад → ҷараён мерасад.

Термоэмиссия: Металлро бо ҷараёни баланд гарм мекунем → бо амперметр хориҷшавии электронҳоро чен мекунем.

Масъалаҳои намунавӣ

Масъала 1 (Содда): Басомади нури таъсиркунанда 7.5×10¹⁴ Гц аст. Кори чидан Aчих = 2 эВ. Оё фотоэффект ба амал меояд?

Ҳал:

Eфотон = hν = 6.63×10⁻³⁴·7.5×10¹⁴ ≈ 4.97×10⁻¹⁹ Ҷ ≈ 3.1 эВ

Азбаски 3.1 эВ > 2 эВ → фотоэффект ба амал меояд

Масъала 2 (Миёна): Энергияи фотон 4 эВ, Aчих = 2.5 эВ. Энергияи кинетикии электрони хориҷшуда чанд аст?

Ҳал:

Eкин = Eфотон - Aчих = 4 - 2.5 = 1.5 эВ

Масъала 3 (Мураккаб): Металли бо φ = 1.1 эВ дар ҳарорати 2000 K ҷойгир аст. Дар формулаи термоэмиссия, афзоиши ҳарорат ба ҷараён чӣ гуна таъсир мерасонад?

Шарҳ:

Мувофиқи j = A·T²·e−φ/kT, агар T зиёд шавад → e−φ/kT зиёд мешавад → ҷараён меафзояд

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Нур бо басомади \(\nu = 1.2×10^{15}\) Гц ба рӯи металл меафтад. Агар Aчи = 3 eV бошад, энергияи кинаматикӣ ва суръати электронро ҳисоб кунед (h = 6.63×10⁻³⁴ Ҷ·с).

2. Барои як металл, агар басомади ҳадди ақал \(\nu_0 = 8×10^{14}\) Гц бошад, Aчи-ро дар eV ҳисоб кунед.

3. Агар фотоэффект мушоҳида нашавад, чӣ гунае фаҳмонда мешавад: нур нарасид ё Aчи калон аст?

4. Дар термоэмиссия, агар ҳарорати сатҳи катод T = 2500 K ва Aчи = 2 eV бошад, ҷараёни эмиссияро бо формулаи Ричардсон ҳисоб кунед (A = 1.2×10⁶ A/m²·K², k = 1.38×10⁻²³ J/K).

5. Тавзеҳ диҳед: чӣ гуна равандҳои фотоэффект ва термоэмиссия дар сохтани лампаҳо ва детекторҳои рӯшноӣ истифода мешаванд?

Ионизатсияи газҳо ва тахлияи мустақил

1. Назарияи асосӣ

Ионизатсия — равандест, ки дар он атом ё молекулаи газ электрон гум мекунад ва ба ион табдил меёбад.

Тахлияи мустақил — гузаштани ҷараён дар газест, ки ба туфайли ионизатсияи худкор ва бе ниёз ба таъсири беруна ба амал меояд.

Моҳият:

  • Газҳо дар ҳолати оддӣ ҷараён нагузаронида наметавонанд
  • Бо таъсири майдони баланд, атомҳо ионизатсия мешаванд
  • Тахлияи мустақил пас аз расидани шиддат ба ҳадди зарурӣ оғоз мешавад
  • Ин раванд асоси кори найчаҳои газӣ, ксенонон ва неон аст

2. Формулаҳои асосӣ

Eион ≥ Wион

Энергияи зарра бояд аз энергияи ионизатсия зиёд бошад

I = I₀·eαd

Қонуни Таунсенд — афзоиши ҷараён дар газ тавассути ионизатсияи тасодуфӣ

Uтахл = Uбарқгир + Uбаҳра

Шиддати тахлия аз ду марҳила иборат аст: ионизатсия ва афзоиш

3. Шарҳ ва мисол:

Агар байни ду электрод газ бошад, ва ба онҳо шиддати баланд дода шавад, пас дар як нуқта, қувваи майдон кофӣ мешавад, ки электронҳо ба атомҳои дигар зарба зада, ионизатсия кунанд → афзоиши лавини ионҳо ва электрони нав → **тахлияи мустақил** оғоз меёбад.

Ин раванд метавонад бо ранг ва садо зоҳир гардад — мисли дурахши найчаҳои неон ё раъду барқ.

4. Таҷрибаи назариявӣ

Таъҷриба: Ду электродро дар найчаи шишагии бо гази камфишор пуркардашуда ҷойгир мекунем. Агар шиддат зиёд карда шавад → майдони электрикӣ электронҳоро метезонад → ба атомҳо зарба зада, ионизатсия ба амал меояд → найча медурахшад.

Натиҷа: таъсири ионизатсия ва тахлияи мустақил мушоҳида мешавад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Барои ионизатсия кардани атом, 13.6 эВ лозим аст. Агар зарра 15 эВ дошта бошад, оё ионизатсия имконпазир аст?

Бале, 15 эВ > 13.6 эВ → ионизатсия рух медиҳад

Масъала 2 (Миёна): Агар I₀ = 0.1 μA ва α = 4 см⁻¹, d = 2 см, ҷараёни баромадро ҳисоб кунед.

I = I₀·eαd = 0.1×e^(4×2) ≈ 0.1×e⁸ ≈ 0.1×2980 ≈ 298 μA

Масъала 3 (Мураккаб): Дар найчае байни электродҳо ионизатсия сурат мегирад. Агар газ фишорро афзоиш диҳад, ба ионизатсия ва тахлия чӣ таъсир мерасонад?

Бо зиёд шудани фишор:

  • Масофаи озоди миёнаи зарраҳо ↓
  • Эҳтимоли бархӯрди бештар ↑ → ионизатсия осонтар мешавад
  • Аммо ҷараёни зиёдтар метавонад ба сӯхтан ё дезинтегратсия оварда расонад

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар зарфи пур аз газ, майдони барқӣ E = 3×10⁴ V/m таъсир мекунад. Агар масофаи байни электродҳо 1 см бошад, фарқи потенсиалро ҳисоб кунед.

2. Тавзеҳ диҳед: чӣ гуна ионизатсияи аввал ва зарбаҳои ионии дуюмӣ метавонанд ба тахлияи мустақил оварда расонанд?

3. Дар муҳити гази ҳалим, эҳтимолияти зарбаи ионӣ барои ҳар 1 мм ҳаракати электрон 0.2 мебошад. Дар масофаи 5 мм, чанд зарбаи миёна интизор аст?

4. Агар барои ташаккули тахлияи мустақил заряди ҳадди ақал 5 нКл лозим бошад ва ҳар як электрон бо энергия 15 eV зарба занад, чанд электрон лозим аст?

5. Тавзеҳ диҳед: чӣ гуна фишори газ, таркиб ва масофаи байни электродҳо ба шиддати ҳадди ақали тахлия таъсир мерасонанд (бо истинод ба қонуни Пашен)?

Плазма ва хосиятҳои он

1. Назарияи асосӣ

Плазма — ҳолати моддаест, ки дар он атомҳо пурра ё қисман ионизатсия шудаанд ва электронҳо озодона ҳаракат мекунанд. Ин муҳит ҳам заряди мусбат ва ҳам заряди манфӣ дорад ва ба майдонҳои электрикӣ ва магнитӣ ҳассос аст.

Хусусиятҳои асосӣ:

  • Плазма аз ионҳо ва электронҳои озод иборат аст
  • Он ҳамчун муҳити гузаронандаи ҷараён рафтор мекунад
  • Хусусиятҳои коллективӣ дорад — таъсири мутақобила байни зарраҳо
  • Дар табиат: офтоб, раъду барқ, полярӣ, оташдонҳои ядроӣ

2. Формулаҳои асосӣ

ne ≈ ni

Шартҳои нейтралии барқӣ (заряди умумӣ ≈ 0)

fp = √(n·e² / (ε₀·m))

Басомади плазма (частота колебаний электронов)

λD = √(ε₀·kT / (n·e²))

Дарозии Дебай — масофаи таъсири экранизасия

3. Шарҳ ва тавзеҳ:

Плазма мисли гази супер-ионшуда аст, аммо бо хосиятҳои махсус: он метавонад мавҷҳо, резонанс, ва ҷараёнҳои мураккабро дастгир кунад. Дар плазма бар хилофи гази оддӣ, зарраҳо бо майдонҳо ва якдигар мутақобила мекунанд. Аз ин рӯ, он “муҳити фаъол” ҳисобида мешавад.

Дар реакторҳои ядроӣ (мисли токамак), плазма барои ба амал овардани реаксияи термоядроӣ истифода мешавад.

4. Таҷрибаи назариявӣ

Таъҷриба: Агар лампа ё найчаи пуршуда бо гази камфишорро ба шиддати баланд пайваст кунем, дар дохили он плазма ба вуҷуд меояд ва ба ранги муайян медурахшад (масалан: неон — сурх, аргон — кабуд).

Хулоса: Зарраҳои газ ионизатсия шуда, ба муҳити плазмавӣ табдил меёбанд.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Дар муҳити плазма n = 10¹⁸ м⁻³. Оё он нейтралии барқиро нигоҳ медорад?

Агар ne ≈ ni, яъне зичии электронҳо ва ионҳо баробар бошад, он гоҳ муҳит нейтрал аст.

Масъала 2 (Миёна): Агар T = 10000 K, n = 10¹⁶ м⁻³, λD-ро ҳисоб кунед.

λD = √(ε₀·kT / n·e²)

Пас аз иваз кардани ададҳо, натиҷаи шумораӣ ҳосил мешавад (тақр. дар сатҳи 10⁻⁴ м)

Масъала 3 (Мураккаб): Чаро плазма метавонад мавҷҳоро нигоҳ дорад, вале гази оддӣ не?

Плазма дорои зарраҳои озоди заряднок аст, ки бо майдонҳои барқӣ ва магнитӣ мутақобила мекунанд. Газ чунин зарраҳоро надорад, бинобар ин он мавҷҳои электромагнитиро нигоҳ дошта наметавонад.

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Тавзеҳ диҳед: плазма чӣ гуна муҳити ҷисмонӣ аст ва чӣ гуна аз гази оддӣ фарқ мекунад?

2. Агар зичии электронҳо \( n = 10^{18} \, \text{m}^{-3} \) бошад, радиуси Дебайро бо истифода аз формулаи \[ r_D = \sqrt{\frac{\varepsilon_0 k T}{n e^2}} \] барои \( T = 10^4 \, K \) ҳисоб кунед.

3. Басомади плазмаро барои зичии \( n = 10^{18} \, \text{m}^{-3} \) бо формулаи \[ \omega_p = \sqrt{\frac{n e^2}{\varepsilon_0 m_e}} \] ҳисоб кунед.

4. Чаро плазма метавонад мавҷҳои электромагнитиро инъикос кунад ва дар кадом шароит аз худ намекунад?

5. Дар муҳити плазмаи термоядро, зарраҳои электрон ва ион бо ҳарорати гуногун ҳастанд. Шарҳ диҳед, чаро тавозуни ҳарорат душвор ба даст меояд.

Индуктсияи магнитӣ ва вектори он

1. Назарияи асосӣ

Индуктсияи магнитӣ — бузургии векториест, ки қувваи таъсири майдони магнитӣ ба заряди ҳаракаткунанда ё ноқилро тавсиф мекунад. Он нишон медиҳад, ки дар кадом самт ва бо кадом қувват майдони магнитӣ амал мекунад.

Хусусиятҳои асосӣ:

  • Бо вектор B ишора мешавад
  • Амал мекунад ба заряди ҳаракаткунанда (қувваи Лоренц)
  • Амал мекунад ба ноқили бо ҷараён (қувваи Ампер)
  • Самти вектор B бо қоидаи бурандаи рост муайян мешавад

2. Формулаҳои асосӣ

F = q·v×B

Қувваи Лоренц — таъсири майдони магнитӣ ба заряди ҳаракаткунанда

F = I·l×B

Қувва ба ноқили бо ҷараён — қонуни Ампер

B = μ₀·I / (2π·r)

Индуктсия дар нуқтаи наздик ба ноқили рост бо ҷараён

3. Шарҳ ва тавзеҳ:

Вектори B ҳам самт ва ҳам андозаи таъсири магнитиро нишон медиҳад. Агар заряди мусбат бо суръат v ба майдон ворид шавад, вектори қувва ба самте мебарояд, ки ба v × B мутобиқ аст (қоидаи се ангушт: v = ангушт, B = каф, F = ангушти калон).

Шумораи хатҳои майдон ва зичии онҳо низ бо B тавсиф мешавад.

4. Таҷрибаи назариявӣ

Таъҷриба: Ноқили ростро ба ду пойдевори металлии ҳаракаткунанда мегузорем ва ба он ҷараён мегузаронем. Ноқил дар майдони магнитии байни қутбҳои магнит ҷойгир аст. Вақте ҷараён дода мешавад → ноқил ҳаракат мекунад → нишон медиҳад, ки B ба он қувва таъсир мекунад.

Натиҷа: Самт ва бузургии B таъсири воқеӣ доранд.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряди 1 μC бо суръати 1000 м/с ба майдони B = 0.2 T бо самти перпендикуляр ворид мешавад. Қувваи таъсиркунанда чанд аст?

F = q·v·B = 1×10⁻⁶ · 1000 · 0.2 = 2×10⁻⁴ N

Масъала 2 (Миёна): Ноқили бо ҷараёни 5 A ва дарозии 0.3 м дар майдони B = 0.1 T ҷойгир аст. Қувваи таъсиркунанда чӣ қадар аст?

F = I·l·B = 5·0.3·0.1 = 0.15 N

Масъала 3 (Мураккаб): Дар нуқтае дар назди ноқили рост бо ҷараёни 10 A ва дар фосилаи 5 см, B-ро ҳисоб кунед.

B = μ₀·I / (2π·r) = (4π×10⁻⁷ · 10) / (2π·0.05) = 4×10⁻⁵ / 0.1 = 4×10⁻⁴ T

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди 2 μC бо суръати 5000 м/с ба самти перпендикуляр ба майдони B = 0.2 Tл ҳаракат мекунад. Қувваи Лоренцро ҳисоб кунед.

2. Агар қувваи Лоренц F = 4×10⁻⁴ Н, заряд q = 1.6×10⁻¹⁹ Кл ва суръат v = 2×10⁶ м/с бошад, B-ро ҳисоб кунед.

3. Ноқили рост бо ҷараёни 5 А дар 10 см дурӣ нуқтае эҷоди майдон мекунад. B-ро дар ин нуқта бо формулаи ноқили рост ҳисоб кунед.

4. Шарҳ диҳед: чӣ гуна самти вектори B бо истифода аз қоидаи бурандаи рост (қоидаи дасти рост) муайян карда мешавад.

5. Тавсиф диҳед: фарқи асосӣ байни вектори шадидияти майдони электрикӣ E ва вектори индуктсияи магнитӣ B дар табиати таъсир ва тарзи ченкунӣ.

Қонуни Био–Савар–Лаплас

1. Назарияи асосӣ

Қонуни Био–Савар–Лаплас вектори майдони магнитии тавлидшуда аз тарафи як элементи ҷараёнро дар нуқтаи мушаххас ҳисоб мекунад. Он вектори индуктсияи магнитӣ-ро тавассути масофа, самт ва бузургии ҷараён муайян мекунад.

Моҳият:

  • Майдони магнитӣ B ба I (ҷараён) мутаносиб аст
  • Баръакси мутаносиб ба квадрати масофа (r²)
  • Самти B тавассути қоидаи бурандаи рост муайян мешавад
  • Қонун барои ҳисобкунии B дар ҳар нуқта атрофи ҷараён истифода мешавад

2. Формулаи асосӣ

d𝐁 = (μ₀ / 4π) · (I·d𝐥 × 𝐫̂) / r²

Формулаи вектории Био–Савар–Лаплас

𝐁 = ∫(μ₀ / 4π) · (I·d𝐥 × 𝐫̂) / r²

Барои ҷараёни пурра — тавассути интеграл ҳисоб мешавад

3. Шарҳ ва тавзеҳ:

Вектор d𝐁 — ҳиссаи хурди майдон, ки аз элементи кӯтоҳи ноқил ба вуҷуд меояд. Самти он аз вектори d𝐥 × 𝐫̂ бармеояд (яъне, перпендикуляр ба ҳам d𝐥 ва r̂).

Ин формула ҳам барои ноқили рост, ҳам ҳалқаи ҷараён, ва ҳам сохторҳои мураккаб татбиқ мешавад.

4. Таҷрибаи назариявӣ

Таъҷриба: Дар атрофи ноқили рост, ҷараёни доимӣ мегузаронем ва бо компас самти майдонро месанҷем. Хатҳои майдон даврашакл ва перпендикуляр ба ноқил мешаванд — мувофиқи қонуни Био–Савар–Лаплас.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Самти майдон дар атрофи ноқили ростро тавсиф намо. Агар ҷараён ба боло равад, самти майдон дар тарафи чапи ноқил чӣ гуна аст?

Аз рӯи қоидаи бурандаи рост: самти майдон дар тарафи чап → ба дохили саҳифа равона шудааст.

Масъала 2 (Миёна): Ҷараёни 3 A тавассути ноқили рост мегузарад. Дар нуқтаи 10 см аз ноқил, B-ро ҳисоб кунед.

B = μ₀·I / (2π·r) = 4π×10⁻⁷ · 3 / (2π·0.1) = 6×10⁻⁷ / 0.2 = 3×10⁻⁶ T

Масъала 3 (Мураккаб): Дар маркази ҳалқа бо радиуси R = 0.05 м ва ҷараёни 2 A, майдони B чанд аст?

B = μ₀·I / (2R) = (4π×10⁻⁷·2)/(2·0.05) = 8π×10⁻⁷ / 0.1 = 8π×10⁻⁶ ≈ 2.5×10⁻⁵ T

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Шарҳ диҳед: қонуни Био–Савар–Лаплас чӣ гуна вобастагии майдони магнитиро аз ҷараён ва мавқеи нуқтаи мушоҳида муайян мекунад.

2. Барои ноқили рост ва дароз бо ҷараёни I = 10 A, дар нуқтае дар фосилаи 5 см аз он, B-ро ҳисоб кунед (\( \mu_0 = 4\pi × 10^{-7} \)).

3. Тавсиф диҳед: чӣ гуна самти \(\vec{B}\) бо истифода аз \(\vec{I} \times \vec{r}\) ё қоидаи дасти рост муайян карда мешавад.

4. Барои ноқили нимдоира бо радиуси 0.1 м ва ҷараёни 2 A, майдони магнитиро дар марказ ҳисоб кунед.

5. Чаро майдони магнитии ноқили печдаро метавон бо ҳамгироии векторӣ ҳисоб кард ва чӣ гуна ин бо симои қонуни Ампер алоқаманд аст?

Қонуни Ампер ва таъсири байни ҷараёнҳо

1. Назарияи асосӣ

Қонуни Ампер нишон медиҳад, ки ду ноқили рост ва дароз, ки бо ҷараёнҳои доимӣ пур шудаанд, ба якдигар қувваи магнитӣ таъсир мекунанд.

Агар ҷараёнҳо ҳамсамт бошанд — ноқилҳо якдигарро ҷалб мекунанд. Агар мухолифсамт бошанд — якдигарро тела медиҳанд.

Моҳият:

  • Майдони магнитӣ аз ҷараёни аввал тавлид мешавад
  • Он ба ҷараёни дуюм қувваи Ампер мерасонад
  • Қувваи таъсир баръакси мутаносиб ба масофа байни ноқилҳо
  • Ин раванд асоси таърифи 1 ампер аст

2. Формулаи асосӣ

F / l = (μ₀ / 2π) · (I₁·I₂ / r)

Қувваи байни ду ноқили бо ҷараён, дар масофаи r ва дарозии l

F = I·l×B

Формулаи умумии Ампер дар майдони B

1 A = ҷараёне, ки дар ду ноқили беохир қувваи 2×10⁻⁷ Н/м тавлид мекунад

3. Шарҳ ва тавзеҳ:

Қонуни Ампер собит мекунад, ки майдони магнитӣ на танҳо таъсир мекунад, балки манбаи қувва мебошад. Таъсири байни ҷараёнҳо дар асоси майдони тавлидшуда аз тарафи яке ва таъсир ба дигаре сурат мегирад.

Ин қонуни физикӣ ҳам аз лиҳози амалӣ (моторҳо, электромагнитҳо), ҳам аз лиҳози таърифии воҳидҳо (ампер) хеле муҳим аст.

4. Таҷрибаи назариявӣ

Таъҷриба: Ду ноқили ростро параллел мегузоранд ва ба онҳо ҷараёнҳои ҳамшакл медиҳанд. Ноқилҳо бо қувваи ночизи магнитӣ якдигарро мекашанд. Агар ҷараёнҳо муқобил бошанд, ноқилҳо якдигарро тела медиҳанд. Ин қувваро метавон бо лавҳаи бармеёфта мушоҳида кард.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Ду ноқили дароз бо ҷараёнҳои 1 A дар масофаи 1 см ҷойгиранд. Қувваи байни онҳо дар як метр чанд аст?

F/l = (μ₀ / 2π) · (I₁·I₂ / r) = (4π×10⁻⁷ / 2π)·(1·1 / 0.01) = 2×10⁻⁷ / 0.01 = 2×10⁻⁵ Н/м

Масъала 2 (Миёна): Агар I₁ = 3 A, I₂ = 5 A, r = 2 см, қувваи байни ноқилҳоро дар 1 м дарозӣ ёбед.

F/l = (μ₀ / 2π) · (I₁·I₂ / r) = (2×10⁻⁷) · (15 / 0.02) = 2×10⁻⁷ · 750 = 1.5×10⁻⁴ Н/м

Масъала 3 (Мураккаб): Ду ноқили бо I = 2 A ҷараён дар ҳолати мутавозин овезонанд. Масофаи байни онҳо = 5 мм. Агар қувва F = 1×10⁻⁵ Н/м бошад, I чанд бояд бошад?

Аз формула:

F/l = (μ₀ / 2π) · (I² / r)

I² = (F·2π·r) / μ₀ → I = √[(1×10⁻⁵·2π·0.005) / (4π×10⁻⁷)] ≈ √(2.5×10⁻⁷ / 4×10⁻⁷) = √0.625 ≈ 0.79 A

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ду ноқили рост ва параллел бо ҷараёнҳои I₁ = 4 A ва I₂ = 6 A дар масофаи 5 см ҷойгиранд. Агар дарозии минтақаи мушоҳида 10 см бошад, қувваи таъсирро ҳисоб кунед (\( \mu_0 = 4\pi×10^{-7} \)).

2. Тавсиф диҳед: чӣ гуна самти қувва байни ноқилҳо вобаста ба самти ҷараён муайян мешавад (қувваи кашиш ё тела)?

3. Барои ноқилҳои параллел бо ҷараёнҳои муқобил, I₁ = 10 A ва I₂ = -10 A, масофа 2 см ва дарозӣ 15 см, қувваро ҳисоб кунед. Оё онҳо якдигарро тела медиҳанд ё мекашанд?

4. Чаро қонуни Ампер барои таърифи ампер (воҳиди ҷараён) асоси амалӣ ҳисоб мешавад?

5. Дар чӣ гуна ҳолат таъсири байни ҷараёнҳо на фақат механикӣ, балки низ электромагнитӣ ҳисобида мешавад (бо мисолҳои трансформатор ё ҳалқаҳои ҷорӣ)?

Таъсири майдони магнитӣ ба заряди ҳаракаткунанда

1. Назарияи асосӣ

Агар заряди электрикӣ бо суръат v ба майдони магнитӣ ворид шавад, он қуввае эҳсос мекунад, ки қувваи Лоренц ном дорад. Ин қувва ба самти ҳаракат ва самти майдон перпендикуляр мебошад ва ҳаракати зарраро ба шакли каҷ ё даврӣ тағйир медиҳад.

Хусусиятҳои таъсир:

  • Қувваи Лоренц танҳо ба зарраҳои ҳаракаткунанда таъсир мекунад
  • Қувва перпендикуляр ба ҳам v ва B аст
  • Азбаски қувва перпендикуляр аст → кори физикӣ намекунад (энергия намебарорад)
  • Самти ҳаракат ба траекторияи даврӣ ё спиралӣ мубаддал мешавад

2. Формулаҳои асосӣ

𝐅л = q·𝐯 × 𝐁

Формулаи вектории қувваи Лоренц

F = |q|·v·B·sinθ

Андозаи қувва — вобаста ба кунҷи байни v ва B

R = m·v / (|q|·B)

Шакли ҳаракат — радиуси даври зарраи дар майдон ҳаракаткунанда

3. Шарҳ ва тавзеҳ:

Агар зарра бо суръат v ба майдони магнитии B ворид шавад ва ин ду самти перпендикуляр дошта бошанд, он гоҳ зарра ба самти каҷ ҳаракат мекунад. Ин ҳаракат даврагӣ аст, зеро қувва ҳамеша ба самти марказ равона мебошад.

Ин асоси кори масс-спектрометрҳо, трубаи Крукс, циклотронҳо, ва дигар дастгоҳҳои плазмавӣ мебошад.

4. Таҷрибаи назариявӣ

Таъҷриба: Дар найчаи шишагӣ, ки дохилаш вакуум ва як гази суст дорад, аз катод ба анод электрони озод партоб мекунем. Майдони магнитӣ таъсир мекунад ва траекторияи электрон ба шакли каҷ ё даврӣ табдил меёбад. Инро бо экран мушоҳида кардан мумкин аст.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Заряди 1 μC бо суръати 500 м/с ба майдони магнитии B = 0.2 T ворид мешавад. Агар кунҷи байни v ва B = 90°, қувваи таъсиркунанда чанд аст?

F = q·v·B·sin(90°) = 1×10⁻⁶ · 500 · 0.2 = 1×10⁻⁴ N

Масъала 2 (Миёна): Зарра бо m = 1×10⁻²⁷ кг, q = 1.6×10⁻¹⁹ Кл ва v = 2×10⁶ м/с ба майдони B = 0.5 T ворид мешавад. Радиуси траектория чанд аст?

R = m·v / (q·B) = (1×10⁻²⁷·2×10⁶) / (1.6×10⁻¹⁹·0.5) ≈ 2.5×10⁻² м = 2.5 см

Масъала 3 (Мураккаб): Агар суръати зарра ба самти B параллел бошад, таъсири майдон ба он чӣ гуна мешавад?

Ҳангоми параллел будани v ва B → sin(θ) = 0 → F = 0 → ягон таъсир нест. Зарра бо ҳаракати рост идома медиҳад.

Вазифаи хонагӣ

Масъалаҳо барои ҳал

2. Агар самти ҳаракати заряд ба майдон кунҷи 60° дошта бошад, бо ҳамон шартҳои масъалаи 1, қувваро бо \(\sin\theta\) ҳисоб кунед.

3. Заряди манфӣ дар майдони магнитӣ бо самти муайян ҳаракат мекунад. Самти таъсири қувва чӣ гуна аст? Қонуни Лоренцро бо қоидаи дасти чап шарҳ диҳед.

4. Заряди +2μC бо суръати 3×10⁵ м/с дар майдони 0.2 Тл гардиши доиравӣ иҷро мекунад. Радиуси траекторияро ҳисоб кунед (m = 1×10⁻⁶ кг).

5. Шарҳ диҳед: чӣ гуна ҳаракати заряди мусбат ва манфӣ дар майдони якхела траекторияҳои гуногун доранд ва чӣ гуна ин дар таҷҳизотҳо истифода мешавад (масалан, спектрометр, телевизорҳои кӯҳна).

Ҳаракати зарраҳо дар майдони магнитӣ

1. Назарияи асосӣ

Ҳангоми ворид шудани заряди ҳаракаткунанда ба майдони магнитии якхела, он таҳти таъсири қувваи Лоренц қарор мегирад ва ҳаракати он ба шакли каҷ, даврӣ, ё спиралӣ табдил меёбад.

Навъи ҳаракат аз самти суръат ва майдон вобаста аст:

  • v ⟂ B → **ҳаракат даврӣ (доиравӣ)**
  • v ‖ B → **ҳаракат рост** (таъсир нест)
  • v кунҷӣ бо B → **ҳаракат спиралӣ (винтовой)**

2. Формулаҳои асосӣ

𝐅л = q·𝐯 × 𝐁

Қувваи Лоренц — самти қувва ҳамеша ба ҳаракат перпендикуляр аст

R = m·v / (|q|·B)

Радиуси траекторияи даврӣ (гардиш)

T = 2π·m / (|q|·B)

Давраи гардиши зарра (вобаста ба масса ва заряд)

f = 1 / T = |q|·B / (2π·m)

Басомади гардиш (частота циклотронӣ)

3. Шарҳ ва тавзеҳ:

Агар зарра ба майдон бо кунҷи 90° ворид шавад, ҳаракаташ **гардиши доиравӣ** мешавад. Қувваи Лоренц онро ба марказ мекашад ва мисли қувваи марказҷазб амал мекунад.

Агар кунҷ байни v ва B хурдтар бошад, зарра ба шакли спиралӣ ҳаракат мекунад — ба самти B ҳаракат мекунад, дар ҳоле ки дар атрофи он давр мезанад.

Ин равандҳо дар физикаи плазма, фазонавардӣ ва адронҳо таҳқиқ мешаванд.

4. Таҷрибаи назариявӣ

Таъҷриба: Дар найчаи вакуумӣ, электрони суръатнок ба майдони магнитӣ ворид карда мешавад. Бо экран ё детектор мушоҳида мешавад, ки роҳи он ба шакли доира ё спиралӣ табдил ёфтааст — вобаста ба самт ва қуввати B.

Ин асоси кори циклотрон ва масс-спектрометр мебошад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Зарраи бо q = 1.6×10⁻¹⁹ Кл ва v = 2×10⁶ м/с ба B = 0.1 Т ворид мешавад (v ⟂ B). Радиуси траектория чӣ қадар аст? (m = 9.1×10⁻³¹ кг)

R = m·v / (q·B) = (9.1×10⁻³¹ · 2×10⁶) / (1.6×10⁻¹⁹ · 0.1) ≈ 1.14×10⁻² м = 1.14 см

Масъала 2 (Миёна): Зарра дар майдони 0.2 Т бо суръати 1×10⁷ м/с бо кунҷи 60° ворид мешавад. Траектория чӣ гуна хоҳад буд?

Ҳиссаи v ба самти майдон → ҳаракати рост, ҳиссаи v ⟂ B → ҳаракати даврӣ → Ҳаракат **спиралӣ (винтовой)** мешавад.

Масъала 3 (Мураккаб): Зарраи номаълум бо m = 2×10⁻²⁷ кг ва q = 1.6×10⁻¹⁹ Кл дар майдони B = 0.2 Т гардиш мекунад. Агар радиуси он 5 см бошад, суръатро ёбед.

v = q·B·R / m = (1.6×10⁻¹⁹·0.2·0.05) / 2×10⁻²⁷ = 1.6×10⁻²⁰ / 2×10⁻²⁷ = 8×10⁶ м/с

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Заряди +1.6×10⁻¹⁹ Кл бо массаи 9.1×10⁻³¹ кг ва суръати 2×10⁶ м/с ба самти перпендикуляр ба майдони B = 0.01 Тл ворид мешавад. Радиуси траекторияро ҳисоб кунед.

2. Бо шартҳои масъалаи 1, даври як гардиш (T) ва басомади гардиш (f)-ро ҳисоб кунед.

3. Агар зарра ба майдон зери кунҷи α ворид шавад, он ҳаракати спиралӣ иҷро мекунад. Шарҳ диҳед, чӣ гуна радиус ва қадами спирал аз α вобаста аст.

4. Заряди манфӣ ба майдон ворид мешавад. Бо истифода аз қоидаи дасти чап самти ҳаракат ва гардишро тавсиф намоед.

5. Чаро ҳаракати зарраҳо дар майдони магнитӣ асоси кори циклотронҳо, спектрометрҳо ва плазмафокусҳо мебошад? Бо мисол тавзеҳ диҳед.

Падидаи Холл

1. Назарияи асосӣ

Падидаи Холл — ин пайдошавии шиддати электрикӣ (шиддати Холл) дар паҳлӯи гузаронандаи ҷараён аст, вақте ки он ба майдони магнитии перпендикуляр ҷойгир карда мешавад.

Моҳият:

  • Дар дохили гузаранда, электронҳо (ё сӯрохҳо) ҳаракат мекунанд
  • Майдони магнитӣ ба онҳо қувваи Лоренц мерасонад
  • Электронҳо ба як канор тела дода мешаванд → фарқи потенсиал ба вуҷуд меояд
  • Ин фарқ — **шиддати Холл (UH)** ном дорад

2. Формулаҳои асосӣ

UH = (B·I) / (n·q·d)

Шиддати Холл вобаста ба B — майдони магнитӣ, I — ҷараён, n — зичии заряд, d — ғафсӣ

EH = UH / w

Шиддати барқии дарунӣ дар паҳлӯи гузаранда

Fл = q·v·B = q·EH

Тавозуни байни қувваи Лоренц ва қувваи электрикӣ

3. Шарҳ ва тавзеҳ:

Падидаи Холл имконият медиҳад, ки мо:

  • Навъи зарраҳои гузаронанда (электрон ё сӯрох)
  • Зичии зарядҳои ҳаракаткунанда
  • Қутбнокии материал (n-тип ё p-тип)

Шиддати Холл дар санҷишгоҳҳо, датчикҳо ва ҳатто смартфонҳо истифода мешавад (масалан, сенсори наздикшавӣ).

4. Таҷрибаи назариявӣ

Таъҷриба: Ба як нимноқили тунук (чипи Холл) ҷараён мегузаронем ва онро дар майдони магнитӣ ҷойгир мекунем. Дар паҳлӯи он шиддати хурде ба вуҷуд меояд — ин шиддати Холл аст. Агар самти майдон ё ҷараён иваз шавад — самти шиддати Холл низ тағйир меёбад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Ҷараёни 2 A аз варақи нимноқил мегузарад. Агар B = 0.5 Т, n = 10²² м⁻³, d = 1 мм, q = 1.6×10⁻¹⁹ Кл — UH чанд аст?

UH = B·I / (n·q·d) = 0.5·2 / (10²²·1.6×10⁻¹⁹·1×10⁻³) ≈ 0.625 мкВ

Масъала 2 (Миёна): Агар шиддати Холл манфӣ бошад, чӣ навъи гузаронанда аст?

Агар UH < 0 → гузариш бо электронҳо (n-тип)

Масъала 3 (Мураккаб): Дар таҷриба, агар ҷараён афзоиш ёбад, шиддати Холл чӣ тавр тағйир меёбад? Агар B низ зиёд шавад?

  • UH ~ I → агар I зиёд шавад → UH ↑
  • UH ~ B → агар B зиёд шавад → UH ↑
  • n, q, d — константа бошад → афзоиши UH хаттӣ аст

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар ноқиле бо паҳнои 2 мм, ҷараёни 5 A ва майдони магнитии 0.3 Tл, шиддати Холл UХ-ро ҳисоб кунед, агар зичии заряд n = 8×10²⁸ м⁻³ бошад (заряд = 1.6×10⁻¹⁹ Кл).

2. Шарҳ диҳед: чӣ гуна самти шиддати Холл вобаста ба навъи заряд (электрон/позитрон) тағйир меёбад?

3. Агар UХ = 5 μV, I = 1 A, B = 0.2 Tл ва паҳнои лавҳа 1 мм бошад, зичии зарядро ҳисоб кунед.

4. Дар як таҷриба муайян карда шуд, ки шиддати Холл манфӣ аст. Хулоса бароред: кадом навъи зарраҳо дар интиқоли ҷараён иштирок мекунанд ва чаро?

5. Барои чӣ падидаи Холл дар таҷҳизоти нимноқилӣ (сенсорҳои Холл) ва муайянкунии зичии заряд дар плазма ё металлҳо истифода мешавад?

Теоремаи Гаусс барои майдони магнитӣ

1. Назарияи асосӣ

Теоремаи Гаусс барои майдони магнитӣ мегӯяд, ки **хатиҳои майдони магнитӣ ҳамеша пӯшидаанд** ва майдон **манбаи (заряди магнитӣ)** надорад. Яъне, хатиҳои майдон аз ягон нуқта оғоз ва дар нуқтаи дигар ба охир намерасанд — онҳо ҳалқашакланд.

Моҳиятҳо:

  • Дар табиат монополҳои магнитӣ мавҷуд нестанд
  • Хатиҳои майдони магнитӣ — ҳамеша пӯшида
  • Флюкси магнитӣ аз ҳар сатҳи пӯшида ҳамеша **баробар ба сифр** аст

2. Формулаи асосӣ

∯𝐁 · d𝐒 = 0

Интеграли флюкси вектори B (индуктсияи магнитӣ) тавассути сатҳи пӯшида баробар ба сифр аст

div 𝐁 = 0

Формулаи дифференсиалӣ — якҷоя бо муодилаҳои Максвелл

3. Шарҳ ва тавзеҳ:

Ин теорема муқобили **теоремаи Гаусс барои электростатика** аст, ки дар он флюкси майдони E вобаста ба заряди дохилӣ аст. Аммо барои магнит:

  • Флюкси B аз ҳар кураи пӯшида **ҳамеша 0**
  • Азбаски заряди магнитӣ вуҷуд надорад → ∇·𝐁 = 0
  • Майдони магнитӣ танҳо аз ҷараёнҳо ва дигар майдонҳо тавлид мешавад

4. Таҷрибаи назариявӣ

Таъҷриба: Бо гузоштани магнит (масалан, бари магнит) ва пошидани хокаҳои оҳан дар атрофи он, хатҳои майдон муайян мешаванд. Онҳо аз қутби шимолӣ берун меоянд ва ба қутби ҷанубӣ ворид мешаванд — вале **дар дохили магнит идома меёбанд**, яъне ҳалқашакланд.

Натиҷа: флюкси холис = 0.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Дар курае, ки дар дохилаш магнит вуҷуд надорад, флюкси майдони B чанд аст?

Аз рӯи ∯𝐁·d𝐒 = 0 → флюкси майдон дар сатҳи пӯшида ҳамеша 0 аст.

Масъала 2 (Миёна): Агар дар сатҳи нимкураии магнит хатҳои майдон аз боло дохил шаванд ва аз поён бароянд, флюкси умумӣ чӣ гуна хоҳад буд?

Флюкси аз боло ва поён якдигарро бекор мекунанд → флюкси умумӣ = 0

Масъала 3 (Мураккаб): Агар флюкси B = 1.2×10⁻⁴ Вб ба сӯи даромад равона бошад ва 1.2×10⁻⁴ Вб аз тарафи дигар барояд, флюкси умумӣ чанд аст?

Флюкси дохил = −1.2×10⁻⁴ Вб, флюкси баромад +1.2×10⁻⁴ Вб → ΣΦ = 0

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар як сатҳи пӯшида майдони магнитӣ ворид шуда ва дар тарафи дигар баробар баромадааст. Ҷараёни умумии майдонро ҳисоб кунед.

2. Шарҳ диҳед: чаро \(\oint \vec{B} \cdot d\vec{S} = 0\) нишон медиҳад, ки "заряди магнитӣ" вуҷуд надорад.

3. Барои сатҳи сферии радиуси任 R дар майдони якхелаи магнитӣ B = const, флюкси майдонро ҳисоб кунед.

4. Агар дар дохили сатҳи пӯшида як ноқили рост бо ҷараён бошад, флюкси магнитӣ аз рӯи он чӣ қадар аст ва чаро?

5. Тавзеҳ диҳед: чӣ гуна ин теорема бо хати майдони магнитӣ алоқаманд аст ва чаро хати B ҳамеша баста аст?

Кори ҷойивазкунии ноқил дар майдони магнитӣ

1. Назарияи асосӣ

Агар ноқили рост бо дарозии l дар майдони магнитӣ B ва бо суръати v ҷойиваз шавад, дар он ЭҚ (электр-қувваи ҳаракатдиҳанда) ба вуҷуд меояд.

Ин амал **кори майдони магнитӣ** бар зарраҳои ҳаракаткунанда мебошад. Дар натиҷа, зарраҳо дар дохили ноқил ба канорҳо тела дода мешаванд ва фарқи потенсиал ба вуҷуд меояд.

Шартҳо:

  • Ноқил бояд майдонро бурида ҳаракат кунад (v ⟂ B)
  • Натиҷа: ЭҚ индуксионӣ дар канорҳои ноқил
  • Майдони B кори мустақим намекунад, балки қувваро "ба роҳ" равона мекунад

2. Формулаҳои асосӣ

𝜀 = B·l·v·sinθ

ЭҚ-и индуксионӣ дар ноқили ҳаракаткунанда

A = q·𝜀

Кори майдон ба заряди q тавассути ЭҚ

Fл = q·v·B

Қувваи Лоренц ба зарраҳо → ҳаракат ба канор

3. Шарҳ ва тавзеҳ:

Агар ноқили металлии рост дар дохили майдони доимии магнитӣ бо суръати v ҳаракат кунад, электронҳои озод дар он аз сабаби қувваи Лоренц ба як тараф тела дода мешаванд. Ин ба вуҷуд омадани фарқи потенсиал ва кори физикӣ мебошад.

Ин равандро метавон барои сохтани **генераторҳои барқӣ** истифода бурд.

4. Таҷрибаи назариявӣ

Таъҷриба: Дар релсаи U-шакл, ноқили рост (бо ғалтаки ҳаракаткунанда) ҷойгир аст. Агар магнит ба он таъсир расонад ва ноқилро ҳаракат диҳем, дар он шиддати барқӣ чен мешавад — бо ёрии вольтметр. Он бо суръат афзун мешавад.

Натиҷа: ЭҚ ба туфайли кори майдон бар зарраҳои дохили ноқил ба вуҷуд меояд.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Ноқили 0.2 м дар майдони 0.5 Т бо суръати 3 м/с ҳаракат мекунад. ЭҚ чанд аст?

𝜀 = B·l·v = 0.5·0.2·3 = 0.3 В

Масъала 2 (Миёна): Агар ноқил бо кунҷи 60° ба хати майдон ҳаракат кунад, чӣ қадар ЭҚ тавлид мешавад (B = 0.4 T, l = 0.5 м, v = 2 м/с)?

𝜀 = B·l·v·sinθ = 0.4·0.5·2·sin60° ≈ 0.4·0.5·2·0.866 ≈ 0.346 В

Масъала 3 (Мураккаб): Агар ноқили 0.3 м бо суръати 5 м/с ба майдони 0.2 Т ҳаракат кунад ва ба он 0.1 Кл заряд пайваст бошад, чӣ қадар кор анҷом мешавад?

𝜀 = B·l·v = 0.2·0.3·5 = 0.3 В → A = q·𝜀 = 0.1·0.3 = 0.03 Дж

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Ноқили дарозии 0.5 м дар майдони магнитии 0.2 Тл бо суръати 3 м/с перпендикуляр ҳаракат мекунад. ЭҚН-и индуктсионӣ дар ноқилро ҳисоб кунед.

2. Агар заряд q = 1.6×10⁻¹⁹ Кл дар натиҷаи ҳаракат W = 3.2×10⁻¹⁹ Ҷ энергия гирад, ЭҚН чӣ қадар аст?

3. Шарҳ диҳед: чаро кори майдони магнитӣ мустақиман ба тағйири энергия намерасонад, аммо дар натиҷаи ҳаракат ЭҚН индуктсия мешавад?

4. Ноқил бо тӯли 1 м ва суръати 5 м/с дар майдони 0.1 Тл ҷойиваз мешавад. Агар ҷараён баста бошад, чӣ гуна қувваҳои бо майдон алоқаманд ба вуҷуд меоянд?

5. Чаро кори ҷойивазкунии ноқил асоси кори генераторҳои барқӣ мебошад? Намунаи амалӣ (велогенератор ё генератори нерӯгоҳ) биёред.

Таҷрибаҳои Фарадей ва хулосаҳо

1. Назарияи асосӣ

Майкл Фарадей дар солҳои 1831 бо якчанд таҷрибаҳо нишон дод, ки тағйирёбии флюкси магнитӣ дар контури ноқилӣ боиси ба вуҷуд омадани ЭҚ (электр-қувваи ҳаракатдиҳанда) ва ҷараёни барқӣ мегардад.

Хулосаҳои асосии Фарадей:

  • Ҷараён танҳо дар натиҷаи тағйири B, на худи B, ба вуҷуд меояд
  • Шиддати ЭҚ ба суръати тағйири флюкси магнитӣ вобаста аст
  • Самти ҷараёни индуксионӣ аз қонуни Ленц муайян мешавад

2. Формулаи асосии Фарадей

𝜀 = − dΦ / dt

Қонуни Фарадей — ЭҚ ба суръати тағйири флюкси магнитӣ мутаносиб аст

Φ = B · S · cosθ

Флюкси магнитӣ: вобаста ба майдон, сатҳ ва кунҷ

𝜀н = −N·dΦ/dt

Барои печҳои сердаҳана (N – шумораи печҳо)

3. Шарҳ ва тавзеҳ:

Таҷрибаҳои Фарадей нишон доданд, ки:

  • Агар магнити доимӣ ба ҳалқаи ноқил наздик карда шавад → ҷараёни муваққатӣ ба вуҷуд меояд
  • Ҷараён танҳо вақте пайдо мешавад, ки **флюкс (Φ)** тағйир меёбад
  • Агар B ё S ё кунҷи θ тағйир ёбад → Φ тағйир меёбад → ЭҚ пайдо мешавад

Ин раванд — **индуксияи электромагнитӣ** — асоси кори генераторҳо, электромоторҳо, трансформаторҳо ва зарядгирии бесим мебошад.

4. Таҷрибаи Фарадей

Таҷриба: Як ҳалқаи мисиро бо галванометр пайваст мекунем. Агар ба дохили ҳалқа магнитро ворид ё хориҷ кунем, галванометр ҳаракат мекунад. Вақте ки магнит дар ҷояш қарор дорад — ҷараён нест. Танҳо ҳангоми тағйирёбии ҳолати майдон ҷараён индуксия мешавад.

Хулоса: Тағйирёбии флюкс → тавлиди ЭҚ.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар флюкси магнитӣ аз 0.02 Вб то 0.01 Вб дар 0.1 с тағйир ёбад, ЭҚ чанд аст?

𝜀 = − ΔΦ / Δt = −(0.01 − 0.02) / 0.1 = +0.1 В

Масъала 2 (Миёна): Печи дорои 100 виток (N = 100), флюкс дар ҳар як давр 0.005 Вб. Агар тағйир дар 0.2 с рӯй диҳад, ЭҚ чанд мешавад?

𝜀 = −N·ΔΦ / Δt = −100·0.005 / 0.2 = −2.5 В

Масъала 3 (Мураккаб): Дар як таҷриба, майдони B тағйир меёбад бо суръати 0.3 Т/с, сатҳ S = 0.05 м², кунҷи 60°. ЭҚ-ро ҳисоб кунед.

dΦ/dt = d(B·S·cosθ)/dt = S·cosθ·dB/dt = 0.05·cos60°·0.3 = 0.05·0.5·0.3 = 0.0075 В → 𝜀 = −0.0075 В

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Таҷрибаи Фарадейро бо ҳалқа ва магнит тавсиф кунед. Кай ЭҚН ба вуҷуд меояд ва кай не?

2. Агар майдони магнитӣ B аз 0.1 Тл то 0.5 Тл дар 2 сония тағйир ёбад ва сатҳи ҳалқа 0.05 м² бошад, ЭҚН-ро ҳисоб кунед.

3. Барои ҳалқаи бо N = 100 печ, тағйири майдон ΔB = 0.2 Тл дар t = 0.1 с, бо сатҳи як печи 0.01 м², ЭҚН-ро ҳисоб кунед.

4. Шарҳ диҳед: чаро агар магнит дар ҳалқа дарун равад ва баъд берун шавад, самти ҷараён баръакс мегардад?

5. Кадом таҷрибаҳои Фарадей асоси кори таҷҳизоти электромагнитӣ мешаванд (генератор, трансформатор)? Намуна оред.

Қонуни Фарадей барои индуксия

1. Назарияи асосӣ

Қонуни Фарадей мегӯяд, ки агар флюкси магнитӣ, ки тавассути контури ноқил мегузарад, дар вақт тағйир ёбад, дар он ЭҚ-и индуксионӣ пайдо мешавад.

Моҳият:

  • ЭҚ (ε) бо суръати тағйирёбии флюкси магнитӣ мутаносиб аст
  • Самти ЭҚ аз рӯи қонуни Ленц муайян мешавад (бо ишораи “−”)
  • Қонун барои як ё чанд даври ноқил татбиқ мешавад

2. Формулаҳои асосӣ

𝜀 = − dΦ / dt

Барои як даври ноқил — ЭҚ баробар ба суръати тағйири флюкси магнитӣ

𝜀 = − N·dΦ / dt

Барои печ бо N давр

Φ = B · S · cosθ

Флюкси магнитӣ тавассути контур

3. Шарҳ ва тавзеҳ:

Тағйирёбии ҳар кадоме аз унсурҳои флюкс (B — майдон, S — сатҳ, θ — кунҷ) метавонад сабаби пайдоиши ЭҚ шавад:

  • Агар B тағйир ёбад → индуксия
  • Агар S (масалан, coil ҳаракат кунад) → индуксия
  • Агар θ (самти ҳаракат) тағйир ёбад → низ индуксия

Қонуни Фарадей барои фаҳмидани кори генераторҳо, трансформаторҳо, зарядгирии бесим ва тормозҳои магнитӣ муҳим аст.

4. Таҷрибаи назариявӣ

Таҷриба: Печи мисиро ба галванометр пайваст мекунем ва ба наздикии он магнит меорем. Ҳангоми ҳаракати магнит, галванометр ҷараёнро нишон медиҳад. Ҳангоми таваққуфи магнит — ЭҚ нест.

Хулоса: Танҳо **тағйири флюкс** боиси ЭҚ мегардад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Флюкси магнитӣ аз 0.04 Вб то 0.01 Вб дар 0.1 с кам мешавад. ЭҚ чанд аст?

𝜀 = − ΔΦ / Δt = −(0.01 − 0.04) / 0.1 = +0.3 В

Масъала 2 (Миёна): Печ дорои 200 давр аст. Агар флюкс дар ҳар давр аз 0.03 Вб то 0.01 Вб дар 0.2 с тағйир ёбад, ЭҚ чанд мешавад?

𝜀 = −N·ΔΦ / Δt = −200·(0.01−0.03)/0.2 = +20 В

Масъала 3 (Мураккаб): Дар печи бо 500 давр, майдони B аз 0.2 Т то 0.5 Т дар 0.1 с зиёд мешавад. Масоҳати давр = 0.01 м², кунҷ = 0°. ЭҚ-ро ёбед.

ΔΦ = ΔB·S = (0.5 − 0.2)·0.01 = 0.003 Вб →

𝜀 = −N·ΔΦ / Δt = −500·0.003 / 0.1 = −15 В

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар ҳалқае бо N = 50 печ, флюкси магнитӣ аз 0.01 В·с то 0.05 В·с дар 0.1 сония зиёд мешавад. ЭҚН-ро ҳисоб кунед.

2. Агар майдон B = 0.2 Тл дар 0.05 сония ба сифр коҳиш ёбад ва сатҳи ҳалқа 0.02 м² бошад (N = 1), ЭҚН-ро ҳисоб кунед.

3. Шарҳ диҳед: чӣ гуна самти ЭҚН бо қонуни Ленц муайян мешавад ва чаро он зидди тағйирот амал мекунад?

4. Барои ҳалқаи N = 100, ΔΦ = 0.03 В·с ва Δt = 0.2 с, ЭҚН-ро бо ишораи манфӣ ҳисоб кунед. Самти ҷараён чӣ гуна аст?

5. Чаро қонуни Фарадей асоси кори трансформаторҳо ва генераторҳо мебошад? Бо мисолҳои амалӣ шарҳ диҳед.

Ҷараёнҳои Фуко

1. Назарияи асосӣ

Ҷараёнҳои Фуко — ҷараёнҳои электрикӣ мебошанд, ки дар дохили ҷисми гузаронинда (масалан, плитаи металлӣ) дар натиҷаи тағйирёбии флюкси магнитӣ индуксия мешаванд. Онҳо дар дохили ҷисм ба шакли ҷараёнҳои гирд (даврзананда) пайдо мешаванд.

Хусусиятҳои муҳим:

  • Ин ҷараёнҳо бо қонуни Фарадей индуксия мешаванд
  • Онҳо дар дохили ҷисм ба шакли ҳалқавӣ ҷорӣ мешаванд
  • Ҷараёнҳои Фуко боиси гармшавии ҷисм мешаванд (эффекти гармидиҳӣ)
  • Онҳо метавонанд зарар (талафоти барқӣ) ё манфиат (тормоз, гармидиҳӣ) дошта бошанд

2. Формулаҳои асосӣ

𝜀инд = − dΦ / dt

ЭҚ, ки сабаби пайдоиши ҷараёни Фуко мешавад

PФуко ∝ B² · f² · d² / ρ

Қувваи талафоти гармӣ (тахминӣ), вобаста ба майдон B, басомад f, ғафсӣ d ва муқовимати барқии модда ρ

IФуко = ε / Rлокалӣ

Ҷараёни гирд дар ҳар як ҳалқаи дохилӣ

3. Шарҳ ва тавзеҳ:

Ҷараёнҳои Фуко боиси **талафоти энергия** дар дастгоҳҳо мешаванд, вале метавонанд бо истифодаи пластинаҳои ламинатсияшуда ё **нақшабандии ферромагнитҳо** кам карда шаванд.

Бо ҳамин раванд, инчунин метавон **тормози бесоиш**, **гармкунии индуксионӣ** ва **тафтиши нуқсони металлҳо** (металлошиносӣ) иҷро кард.

4. Таҷрибаи назариявӣ

Таҷриба: Як магнити пурқувватро болои варақи мисӣ ё алюминий ҳаракат медиҳем. Магнит суст мешаваду “тормоз” мекунад — сабаб: ҷараёнҳои Фуко боиси тавлиди майдони муқобил мешаванд (қонуни Ленц).

Хулоса: Ҷараёнҳои Фуко ба тағйирёбии B вокуниш нишон медиҳанд ва майдони зиддииндуксионӣ ба вуҷуд меоранд.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Чаро варақи мисӣ дар майдони тағйирёбандаи магнитӣ гарм мешавад?

Тағйирёбии флюкс → ЭҚ → ҷараёни Фуко → гармшавӣ бар асари муқовимати дохилӣ.

Масъала 2 (Миёна): Чӣ гуна метавон талафоти ҷараёнҳои Фуко-ро дар трансформатор кам кард?

  • Бо истифодаи ядрои **ламинатсияшуда** (пластина-пластина)
  • Бо истифодаи моддаҳои муқовимати баланд
  • Кам кардани басомад

Масъала 3 (Мураккаб): Чаро магнит ҳангоми афтодан ба найчаи мисии амудӣ суст мешавад?

Ҳангоми ҳаракати магнит → флюкси магнитӣ дар девори найча тағйир меёбад → ҷараёнҳои Фуко индуксия мешаванд → онҳо майдони зидди ҳаракат тавлид мекунанд → тормози электромагнитӣ.

Масъалаҳо барои ҳал

2. Дар таҷриба, плитаи мисӣ ба майдони тағйирёбандаи магнитӣ ворид карда мешавад ва гарм мешавад. Чаро?

4. Чаро дар таҷҳизотҳо барои кам кардани таъсири ҷараёнҳои Фуко плитаҳоро ламелӣ (бурида-бурида) месозанд? Бо мисоли трансформатор шарҳ диҳед.

Худиндуксия ва мутақобила

1. Назарияи асосӣ

Худиндуксия — ин падидаест, ки дар он дар дохили , ҳангоми тағйирёбии ҷараёни барқ, флюкси магнитӣ низ тағйир меёбад ва боиси ба вуҷуд омадани **ЭҚ-и индуксионӣ дар ҳамон контур** мешавад.

Индуксияи мутақобила — падидаест, ки ҳангоми тағйир ёфтани ҷараён дар як печ, флюкси магнитӣ дар печи дигар тағйир ёфта, дар он **ЭҚ-и индуксионӣ** ба вуҷуд меояд.

Фарқиятҳо ва умумиятҳо:

  • Худиндуксия — таъсири майдони худ ба худ
  • Мутақобила — таъсири як печ ба печи дигар
  • Ҳарду бо қонуни Фарадей тавсиф мешаванд

2. Формулаҳои асосӣ

𝜀х = −L·(dI / dt)

ЭҚ-и худиндуксия, L — индуктивият

L = N²·μ·S / l

Индуктивият барои печ бо N давр, масоҳати S ва дарозии l

𝜀м = −M·(dI / dt)

ЭҚ-и мутақобила, M — коэффициенти мутақобила

3. Шарҳ ва тавзеҳ:

⮞ Ҳангоми зиёд ё кам шудани ҷараён дар печ, флюкси B тағйир меёбад → ин боиси ЭҚ мегардад, ки самт ва таъсири муқобил дорад (қонуни Ленц).

⮞ Агар ду печ наздик бошанд, B-и якум ба дуюм мерасад → индуксияи мутақобила сурат мегирад (асоси кори **трансформаторҳо**).

L ва M аз хусусиятҳои геометрии печ, моддаи дохилӣ ва шумораи печҳо вобастаанд.

4. Таҷрибаи назариявӣ

Худиндуксия: Агар ҷараён дар печи индуктивӣ тағйир дода шавад, дар ҳамон печ ЭҚ пайдо мешавад ва дурахши лампа (ё партофт дар экран) сабт мешавад.

Мутақобила: Агар як печро ба манбаъ пайваст кунем ва дигареро ба галванометр, бо иваз кардани ҷараён дар якум, ҷараёни муваққатӣ дар дуюм пайдо мешавад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар L = 0.5 Гн ва dI/dt = 2 А/с бошад, ЭҚ-и худиндуксия чанд аст?

𝜀 = −L·dI/dt = −0.5·2 = −1 В

Масъала 2 (Миёна): Печ бо 200 давр, S = 5×10⁻⁴ м², l = 0.1 м, μ = 4π×10⁻⁷. L чанд мешавад?

L = N²·μ·S / l = (200)²·4π×10⁻⁷·5×10⁻⁴ / 0.1 ≈ 2.5×10⁻³ Гн

Масъала 3 (Мураккаб): Агар дар печи якум тағйири ҷараён dI/dt = 3 А/с ва коэфф. мутақобила M = 0.02 Гн бошад, ЭҚ дар дуюм чанд мешавад?

𝜀 = −M·dI/dt = −0.02·3 = −0.06 В

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар пече бо L = 0.2 H, ҷараён дар 0.01 сония аз 0 то 5 А меафзояд. ЭҚН-и худиндуксияро ҳисоб кунед.

2. Агар \(\frac{dI}{dt} = 50\) A/s ва \(\mathcal{E}_\text{х} = 2\) В бошад, индуктивиятро ҳисоб кунед.

3. Дар ду печи наздик M = 0.01 H ва \(\frac{dI_1}{dt} = 100\) A/s. ЭҚН-и мутақобила дар дуюмро ҳисоб кунед.

4. Шарҳ диҳед: чӣ гуна худиндуксия боиси таъхири ҷараён дар печ мегардад ва дар кадом таҷҳизот (лампа, релле) ин дида мешавад?

5. Дар трансформатор, чӣ гуна мутақобила байни ду coil ба истеҳсоли ЭҚН дар coil дуюм мерасонад? Шакли умумии формуларо нависед ва мисол биёред.

Трансформаторҳо ва принсипи кори онҳо

1. Назарияи асосӣ

Трансформатор — дастгоҳест, ки шиддати ҷараёни тағйирёбандаро (AC) ба арзиши дигар (баланд ё паст) табдил медиҳад. Коркарди он бар асоси **қонуни Фарадей** ва **индуксияи мутақобила** мебошад.

Он аз ду печи мисӣ (первичка ва вторичка) иборат аст, ки дар ядрои ферромагнитӣ ҷойгиранд. Ҷараёни тағйирёбандаи AC дар печи аввалӣ флюкси магнитиро тавлид мекунад, ки дар печи дуюм **ЭҚ-и индуксионӣ** ба вуҷуд меорад.

Навъҳо:

  • Баландкунанда: U₂ > U₁
  • Пасткунанда: U₂ < U₁
  • Фаъолият танҳо бо ҷараёни тағйирёбанда (AC)

2. Формулаҳои асосӣ

U₁ / U₂ = N₁ / N₂

Нисбати шиддатҳо ба нисбати даврҳои печҳо мутаносиб аст

I₁ / I₂ = N₂ / N₁

Баръакс мутаносибии ҷараёнҳо (барои идеалии бетаъсирот)

U₁·I₁ = U₂·I₂

Қонуни нигоҳдории энергия (бе талафот)

3. Шарҳ ва тавзеҳ:

Трансформатор боиси тағйири шиддат мегардад, вале **энергияро нигоҳ медорад** (идеалӣ). Агар печи аввалӣ зиёдтар аз дуюм бошад → пасткунӣ, агар баръакс → баландкунӣ.

Барои паст кардани талафоти ҷараёнҳои Фуко, ядро аз **ламинатсия** (пластинаҳои борик) сохта мешавад.

4. Таҷрибаи назариявӣ

Таҷриба: Ба печи аввалии трансформатор манбаи тағйирёбандаи AC пайваст мекунем. Дар печи дуюм бо ёрии вольтметр шиддат чен карда мешавад. Агар N₂ > N₁ → шиддат афзоиш меёбад. Агар N₂ < N₁ → шиддат кам мешавад.

Хулоса: ЭҚ дар печи дуюм тавассути **индуксияи мутақобила** индуксия мешавад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Трансформатор дорои N₁ = 100, N₂ = 200. Агар шиддати вуруд 10 В бошад, шиддати баромад чанд аст?

U₂ = U₁·(N₂ / N₁) = 10·(200 / 100) = 20 В

Масъала 2 (Миёна): Агар U₁ = 220 В, I₁ = 0.5 А, ва N₁/N₂ = 2, I₂ чанд мешавад (бетаъсирот)?

I₂ = I₁·(N₁ / N₂) = 0.5·2 = 1 А

Масъала 3 (Мураккаб): Агар дар як трансформатор қувваи вурудӣ 100 Вт бошад ва талафот 10%, қувваи баромад чанд аст?

P₂ = P₁ · η = 100·0.9 = 90 Вт

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Трансформатор дорои N₁ = 1000 ва N₂ = 100 печ мебошад. Агар шиддати вуруд U₁ = 220 В бошад, шиддати баромадро ҳисоб кунед.

2. Дар трансформатор бо таносуби печҳо 10:1, агар ҷараёни баромад I₂ = 5 A бошад, ҷараёни вурудро I₁ ҳисоб кунед.

3. Шарҳ диҳед: чӣ гуна тағйири ҷараён дар печи аввал боиси индуксияи ЭҚН дар печи дуюм мегардад (қонуни Фарадей + мутақобила).

4. Агар дар трансформатор 5% талафоти нерӯ бошад ва P₁ = 500 Вт бошад, қудрати баромадро (P₂) ҳисоб кунед.

5. Барои чӣ трансформатор танҳо бо ҷараёни тағйирёбанда (AC) кор мекунад ва на бо доимӣ (DC)? Бо шарҳи физикӣ фаҳмонед.

Энергияи майдони магнитӣ

1. Назарияи асосӣ

Майдони магнитие, ки дар натиҷаи ҷараён дар печ (индуктор) ба вуҷуд меояд, метавонад **энергия захира кунад**. Ин энергия ҳамчун энергияи майдони магнитӣ номида мешавад.

Ҳангоми гузариши ҷараён дар печ, тавлиди майдон сурат мегирад ва барои ин кор анҷом дода мешавад. Ин кор ба шакли энергия дар худи майдон нигоҳ дошта мешавад.

Нуктаҳои муҳим:

  • Энергия дар худи майдони магнитӣ захира мешавад
  • Он вобаста ба ҷараён ва индуктивият аст
  • Барои печ ё индуктор, формулаи муайян дорад

2. Формулаҳои асосӣ

W = (1/2)·L·I²

Энергияи захиравии печ бо индуктивият L ва ҷараён I

w = B² / (2·μ)

Зичии энергияи майдон (барои вакуум ё муҳити магнитӣ)

W = ∫(B² / 2μ)·dV

Энергияи умумӣ дар ҳаҷми V

3. Шарҳ ва тавзеҳ:

⮞ Ҳангоми зиёд шудани ҷараён дар печ, майдон тавлид мешавад ва система энергия мегирад.

⮞ Агар ҷараён қатъ шавад, майдон “фурӯ меравад” ва энергия метавонад ба шакли ЭҚ баргардад (масалан, дар схемаи резонӣ).

⮞ Ин формула асоси кори индукторҳо, релеҳо, трансформаторҳо, муҳаррикҳо ва дигар дастгоҳҳои барқӣ мебошад.

4. Таҷрибаи назариявӣ

Таъҷриба: Агар печи индуктор ба лампа пайваст бошад ва манбаи ҷараён якбора канда шавад, лампа як лаҳза дурахш мекунад. Сабаб — энергияи майдони магнитии печ ба ЭҚ табдил меёбад ва дар лампа ба равшанӣ мегузарад.

Хулоса: Печ метавонад энергия захира кунад ва ҳангоми қатъшавӣ онро барорад.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар L = 0.2 Гн ва I = 2 А бошад, энергия дар печ чанд аст?

W = (1/2)·L·I² = 0.5·0.2·4 = 0.4 Дж

Масъала 2 (Миёна): Дар печ бо 400 давр, L = 0.5 Гн ва I = 3 А. Энергияи захирашуда чанд аст?

W = 0.5·0.5·9 = 2.25 Дж

Масъала 3 (Мураккаб): Агар зичии энергияи майдон w = 5 Ҷ/м³ ва ҳаҷми майдон 0.02 м³ бошад, W чанд мешавад?

W = w·V = 5·0.02 = 0.1 Дж

Вазифаи хонагӣ

Масъалаҳо барои ҳал

1. Дар пече бо индуктивияти L = 0.5 H ва ҷараёни I = 4 A, энергияи захирашудаи майдонро ҳисоб кунед.

2. Агар B = 0.3 Tл ва \(\mu_0 = 4\pi×10^{-7}\), зичии энергияро дар фазо ҳисоб кунед.

3. Агар I дар печ аз 0 то 5 A тағйир ёбад, чӣ қадар энергия дар майдон ҷамъ мешавад (L = 1 H)?

4. Шарҳ диҳед: чӣ гуна энергияи магнитӣ ҳангоми хомӯш кардани ҷараён ба гармӣ ё ЭҚН-и худиндуксионӣ табдил меёбад?

5. Тавзеҳ диҳед: чаро печҳо ҳамчун "аккумуляторҳои майдон" истифода мешаванд ва чӣ гуна ин дар реллеҳо ё конденсаторҳои пурқувват дида мешавад?

Муодилаҳои Максвелл барои майдони электромагнитӣ

1. Назарияи асосӣ

Муодилаҳои Максвелл — низоми чор муодилаи асосии электродинамика мебошанд, ки ҳамкории майдонҳои электрикӣ (𝐄) ва магнитӣ (𝐁)-ро бо зарраҳо ва ҷараён тавсиф мекунанд.

Онҳо аз тарафи олими англис Ҷеймс Кларк Максвелл ҷамъбаст шуда, асоси фаҳмиши мавҷҳои электромагнитӣ, кори антеннаҳо, генераторҳо, муҳаррикҳо ва рӯшноӣ мебошанд.

Муодилаҳо дар шакли дифференсиалӣ:

  1. 1. Теоремаи Гаусс барои майдони электрикӣ:
    ∇·𝐄 = ρ / ε₀

    → Заря манбаи майдони электрикӣ аст.

  2. 2. Теоремаи Гаусс барои майдони магнитӣ:
    ∇·𝐁 = 0

    → Майдони магнитӣ монопол (заряди магнитӣ) надорад; хатиҳо ҳамеша пӯшидаанд.

  3. 3. Қонуни Фарадей (индуксияи электромагнитӣ):
    ∇×𝐄 = − ∂𝐁 / ∂t

    → Тағйирёбии майдони магнитӣ → майдони электрикӣ ба вуҷуд меояд.

  4. 4. Қонуни Ампер-Максвелл:
    ∇×𝐁 = μ₀𝐉 + μ₀ε₀ ∂𝐄 / ∂t

    → Ҷараён ва тағйири майдони электрикӣ → майдони магнитӣ тавлид мекунанд.

2. Формулаҳои интегралӣ

  • ∮𝐄·d𝐒 = Q / ε₀ – Флюкси майдони электрикӣ баробар ба заряди дохили сатҳ
  • ∮𝐁·d𝐒 = 0 – Флюкси магнитӣ аз сатҳи пӯшида ҳамеша 0 аст
  • ∮𝐄·d𝐥 = − dΦB / dt – Индуксияи ЭҚ тавассути тағйири B
  • ∮𝐁·d𝐥 = μ₀I + μ₀ε₀ dΦE/dt – Ампер бо ислоҳи Максвелл

3. Шарҳ ва тавзеҳ:

  • Муодилаҳои Максвелл ҳама ҳодисаҳои электромагнитиро тавсиф мекунанд: электростатика, индуксия, мавҷҳо ва паҳншавии рӯшноӣ.
  • Аз ин чор муодила метавон **муодилаи мавҷи электромагнитӣ**-ро ҳосил кард:
∇²𝐄 = μ₀ε₀·∂²𝐄 / ∂t²

→ Ин мавҷи рӯшноӣ бо суръати:

c = 1 / √(μ₀ε₀) ≈ 3×10⁸ м/с

Яъне рӯшноӣ худаш мавҷи электромагнитист!

4. Таҷрибаи назариявӣ

Фаҳмондани муодилаи Фарадей: Агар магнитро ба ҳалқаи мисии пайваст ба галванометр наздик кунем → тағйири майдони магнитӣ → ҷараён (ЭҚ) индуксия мешавад.

Фаҳмондани муодилаи Максвелл: Тағйирёбии майдони электрикӣ (масалан, дар конденсатор) низ метавонад майдони магнитиро ба вуҷуд орад, ҳатто агар ҷараён вуҷуд надошта бошад — ин навоварии Максвелл буд.

5. Масъалаҳои намунавӣ

Масъала 1 (Содда): Агар дар сатҳи пӯшида ∮𝐁·d𝐒 = 0 бошад, чӣ натиҷагирӣ мекунем?

Флюкси майдони магнитӣ ҳамеша 0 аст → манбаи магнитӣ (монопол) вуҷуд надорад → хатиҳои майдон пӯшидаанд.

Масъала 2 (Миёна): Тағйирёбии B = 0.3 Т/с, S = 0.1 м². ЭҚ чанд мешавад?

ε = − dΦ / dt = − S·dB/dt = −0.1·0.3 = −0.03 В

Масъала 3 (Мураккаб): Агар ∂𝐄/∂t = 2×10¹¹ В/м·с, μ₀ = 4π×10⁻⁷, ε₀ = 8.85×10⁻¹², майдони магнитӣ дар муҳити вакуум чӣ қадар афзоиш меёбад?

Аз ∇×𝐁 = μ₀ε₀·∂𝐄/∂t → B = μ₀ε₀·∂𝐄/∂t

= 4π×10⁻⁷ · 8.85×10⁻¹² · 2×10¹¹ ≈ 2.2×10⁻⁷ Т
